# 逸剑风云决（Wandering Sword）— 战斗系统与整体设计调研

> 国产像素武侠 RPG，2023 年 9 月上线，开发商"侠萌游戏"（The Swordman Studio）。
> Steam 特别好评（95% / 34,000+ 评论），是近年国产武侠单机的现象级作品。
> 同 IGN 中国 7 分（"扎实内容掩盖玩法深度不足"），但更新两年后，2025 末版本被资深玩家评为"完全体"。

## 一、游戏概述

| 项 | 内容 |
|---|---|
| 全名 | 逸剑风云决 / Wandering Sword |
| 开发商 | The Swordman Studio（侠萌游戏，国内独立团队）|
| 发行商 | Spiral Up Games |
| 发售 | 2023-09-14（Steam）|
| 平台 | PC（Windows 7/8/10 64bit）|
| 引擎 | Unreal Engine 4 |
| 类型 | 武侠 RPG / 半开放世界 / 半即时战棋 |
| 主线时长 | 20–40h；全成就 90h+；持续更新中 |
| Metacritic | 80 |
| 美术风格 | HD-2D（2D 像素 + 3D 空间 + 现代光影）|
| DLC 节奏 | 2024.03 南疆风云 / 2024.10 烟尘回响 / 2025.06 武家旧事（免费）+ 碧海仙踪（付费）|

**设计目标**：用像素美术 + 经典 JRPG 节奏，做一款**"江湖味浓、有人情味"的中式武侠 RPG**——
> "你不是一个等级 1 的勇者，你是一个 17 岁中毒的少年；
> 江湖里所有人都比你强，你要靠一点机缘、一点选择，
> 慢慢成为可以查清那段恩怨的大侠。"

---

## 二、核心循环

```
开放地图行走（5 大地域 / 75+ 场景）
    ↓
进入战斗 → 半即时格子战
    ↓
战利品 / 武学秘籍 / 心法
    ↓
回客栈休整 → 经脉打通 / 武学升级 / 心法装备
    ↓
推进主线 / 探索支线 / 招募同伴
```

宏观哲学：**"内容堆叠 + 深度叙事 + 蝴蝶效应"**——它不靠玩法深度赢，而靠"江湖丰满 + 选择有重量"赢。

## 三、关键机制详解

### 3.1 半即时制战斗（核心创新）

每个角色头顶有：
- **绿条**：HP
- **黄条**：集气条（行动条）

**集气满即可行动**——但行动模式有两种：

| 模式 | 当某角色行动时 | 队友 | 适用玩家 |
|---|---|---|---|
| **回合模式** | 其他单位**全部暂停** | 玩家**亲自指挥** | 慢思考、爱排兵布阵 |
| **即时模式** | 其他单位**继续集气并行动** | AI 自主作战 | 临场反应、爱躲招对招 |

**两种模式可在游戏设置里随时切换**——这是它最大胆、也是被业界讨论最多的设计。

**战斗指令 6 类**：
| 类 | 真气 | 冷却 | 备注 |
|---|---|---|---|
| 普攻 | 0 | 0 | 兜底输出 |
| 二式 | 有 | 短 | |
| 三式 | 有 | 中 | |
| 绝式 | 多 | 长 | 终结技 |
| 轻功 | 有 | — | 提升集气、特殊位移 |
| 内功（心法）| 有 | — | 回血 / Buff |

每类武学有 **10 级**，刷怪获取经验升级。

### 3.2 武学体系（200+ 秘籍 / 6 大流派）

| 流派 | 风格 | 代表 |
|---|---|---|
| 拳掌 | 防守反击 | 太极拳 |
| 剑法 | 高暴击、十字/直线贯穿 | 独孤九剑 |
| 刀法 | 横排扫荡 | — |
| 棍法 | 周身群体打击 | — |
| 暗器 | 远程偷袭 | — |
| 内功 | 真气/Buff/回血 | — |

**武器精通 5 类**——拳掌 / 刀 / 剑 / 长武器 / 暗器，**可同时修炼**全才路线。

### 3.3 经脉系统 + 心法装备 + 武学点 三轨养成

**核心创新：游戏没有传统"等级 + 经验值"**——属性增长完全靠养成系统：

| 养成层级 | 来源 | 用途 |
|---|---|---|
| **武学点** | 升级当前装备的秘籍 | 提升武功本身的伤害与精通 |
| **经脉点** | 升级装备中的"心法" | 解锁穴位 → +属性 / +暴击/闪避 / 概率再次行动 |
| **武器精通** | 用对应类武器战斗 | 主动技能威力总加成 |

**秘籍来源**：宝箱、任务奖励、**向加入队伍的 NPC 学习**（达到好感度后消耗 50–60 精力学会）

→ "学武功"本身是 RPG 的一部分——比"升级树自动加点"更**契合武侠 IP**。

### 3.4 蝴蝶效应支线 + 20 种结局

**全游戏共 20 种结局**——支线选择在表面无关的章节会通过蝴蝶效应改变最终结局。

**经典案例：武当师兄元启支线**

| 走法 | 后果 |
|---|---|
| 仅按任务指引 | 元启走不出心魔 → 被天魔教唆背刺清霄 → 黑化 → 坏结局 |
| 深入探索（找到断剑、剑法笔记并在关键时刻拿出）| 开导成功 → 元启大战中救李元兴 → 解锁"天涯比邻"等 4 个隐藏成就 |

**设计哲学**：每条支线都有"完美解"——区别于《巫师 3》"血腥男爵"那种"必有遗憾"的设计。
**任务系统存在"误导性"**：跟着标记走必然差体验，**唯有沉浸式探索**才能解锁好装备、好剧情、好结局。

→ 这是一种**对"沉浸度"的考验与奖励机制**——非常硬核。

### 3.5 同伴系统（14 名可招募 + 恋爱）

- 14 名独特武者可招募
- 每人有专属剧情
- 部分可发展恋爱关系
- 队友是**学武功的渠道**——好感度满后消耗精力学他/她的秘籍

### 3.6 HD-2D 美术（最高识别度）

- UE4 引擎制作
- 像素角色 + 3D 场景 + 现代光影
- **5 大地域 / 75+ 场景**：江南水乡、塞外荒漠、山林洞窟、险峻地势、险恶巢穴
- **移轴摄影风滤镜**——前景清晰、远景虚化，"像水墨画"
- 评论原话："纸张做旧风格配色，营造浓郁时代感与武侠意味"

### 3.7 持续更新打造的"完全体"

| 时间 | 更新 |
|---|---|
| 2023.09 | 首发 |
| 2024.03 | 南疆风云 |
| 2024.10 | 烟尘回响 |
| 2025.06 | 武家旧事（免费）+ 碧海仙踪（付费 DLC）|

90h 玩家原话：**"严格意义上是 2025 年后才全部完成"**——首发版与现在版差距巨大。
**新人必装 DLC「武家旧事」**，否则会漏掉大段剧情。

### 3.8 探索奖励系统

- 地图各角落散布**大量宝箱**
- 与 RPG Maker 罐头那种"宝箱只有小垃圾"不同，**本作宝箱可能直接大幅提升属性**
- "藏匿点 + 高奖励"组合产生强烈惊喜感与探索欲望

## 四、为什么好玩（抽象层）

| 体验驱动力 | 具体由什么产生 |
|---|---|
| **武侠味浓** | HD-2D 美术 + 国风音乐 + 经典武侠 IP（武当/少林/丐帮/恶人谷/五毒教）|
| **江湖人情味** | 14 名同伴专属剧情 + 蝴蝶效应支线 + 20 结局 |
| **探索惊喜** | 75+ 场景 + 高品质宝箱 + 隐藏秘籍 |
| **武学养成契合 IP** | 无传统等级，靠武学/经脉/心法成长 |
| **战斗灵活** | 半即时双模式可切换 |
| **沉浸度被奖励** | 仔细探索 → 解锁好结局；这是**软性的"硬核"** |
| **持续更新** | 两年三大版本，开发商真的在打磨 |

## 五、设计缺陷与争议

### 5.1 战斗系统是最大短板（IGN 7 分核心理由）
1. **战斗地图无高低差、无遮挡物**——战术维度单薄
2. **无反击设计、无属性生克**
3. **敌方 AI 几乎不存在**——杂兵只会"攻击离自己最近的敌人"
4. **招式之间无连携、互动**特殊效果
5. **无自动战斗**，反复刷小怪非常折磨
6. 特效简陋，动画种类少，音效糊弄
7. 最终沦为**比拼数值**的游戏

→ **借鉴时必须正面解决这个问题**——不能照搬。

### 5.2 角色朝向只有左右两向
- 理论上必定会被侧击/背击（只能靠 AI 放水平衡）
- 与《大侠立志传》同病
- 对"战术站位"的体验是减分项

### 5.3 内功叠加导致平衡崩坏
- 玩家社区原话："内功叠加致平衡崩坏"
- 高品装备/心法直接覆盖低品，前期辛苦攒的装备很快被淘汰

### 5.4 跑图体验差
- **移动速度慢**，无辅助交通工具，驿站/渡口太少
- 大地图上**几乎无任何互动**——无遇敌、无采集点、无随机事件
- **无完整缩略地图**，无清晰目的地指引——"穿越回 90 年代"
- 鞋子虽有移速副词条，但仅是心理安慰

### 5.5 任务指引的"误导性"是双刃剑
- 沉浸玩家："蝴蝶效应让选择有重量"
- 普通玩家："为什么我跟着任务标记走却拿坏结局？"
- 资深玩家原话："**务必查看最新版攻略**，老攻略会让玩家漏掉关键剧情和支线"——这本质上是**设计强迫玩家攻略化**

### 5.6 生产系统过于简化
- 锻造、制作、烹饪、钓鱼**全部无小游戏过程**——集齐材料点确定即得
- 唯一悬念是高级装备的随机词条
- 素材消耗快，靠装备分解缓解焦虑

### 5.7 难度忽高忽低
- IGN 原话："难度忽高忽低，影响故事流畅度"
- 主因是**养成系统过于自由**——错误搭配心法 = 卡 5h；正确搭配 = 横扫
- 没有"难度自适应"机制

### 5.8 早期版本剧情薄弱
- 90h 评测原话："得益于'南疆风云''烟尘回响'两次更新，弥补了早期人物弧光和剧情深度不足"
- → 这也提示：**长线 RPG 不可能首发完美**，要规划"持续更新"

### 5.9 部分立绘 AI 感
- 美术整体优秀，但部分立绘"AI 感较强"
- 玩家建议："聘请画师为主要角色加一套手绘立绘皮肤"

## 六、对我们项目的启发

### 6.1 直接拿（哲学层）
- ✅ **"无传统等级，靠养成系统"**——契合武侠/中世纪雇佣兵 IP，比"刷级 9 级"更有代入感
  - 我们的"装备 + 武器熟练度 + Trait"已经在这条路上
- ✅ **支线决定结局**——蝴蝶效应支线设计是**剧情有重量**的标准答案
- ✅ **沉浸度被奖励**——但要警惕变成"必查攻略" → 我们的应对是"在游戏内给提示"

### 6.2 直接拿（机制层）
- ✅ **集气条 / 行动条**——确认我们走 TO + 逸剑风云决的"集气条 + 半即时"路线（详见后续 action-bar-systems.md）
- ✅ **武学/招式分类**（普攻/二式/三式/绝式/轻功/心法）——映射到我们已有的"武器关键字 + 技能"系统
- ✅ **武器精通 5 类全才路线**——和我们的"无职业，靠装备决定打法"完全合拍
- 🟡 **同伴学武**——值得保留为"招募即解锁雇佣兵专属技能"
- 🟡 **回合 / 即时双模式**——**先做回合模式**，即时模式作为 Phase 4+ 候选

### 6.3 警示（避开它的坑）
- ❌ **战斗地图无高低差/无遮挡**——我们必须做战术地形（六边形 + 高度差或遮挡）
- ❌ **AI 只追最近**——我们已经规划"敌方 AI 有意图（聚焦/守序/狂战）"
- ❌ **无反击 / 无连携**——夹击（来自 TO）+ 反击（来自 Berwick）必须做
- ❌ **角色仅左右朝向**——我们用六边形 6 朝向，至少机制上不会塌
- ❌ **跑图无互动**——大地图必须有遇敌点、采集点、随机事件
- ❌ **生产无小游戏**——锻造/合成必须有"参与感"，哪怕是简单的 QTE 或选择
- ❌ **任务指引误导性**——蝴蝶效应可以做，但**关键节点要在游戏内有"提示性铺垫"**，不能逼玩家查攻略

### 6.4 真正值得借鉴的"逸剑精神"
1. **"内容堆叠 > 玩法创新"**——75 场景 + 200 秘籍 + 14 同伴 + 20 结局，**靠量与密度撑起世界感**
2. **持续更新打造完全体**——首发不必完美，但要规划长线 DLC 节奏
3. **武侠/江湖味是叙事粘合剂**——我们用中世纪雇佣兵 IP 也要找到自己的"江湖味"（佣兵团 + 联络网 + 任务板？）
4. **HD-2D 是低成本高识别度的美术答案**——值得作为美术风格备选
