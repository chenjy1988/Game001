# 圣女贞德（Jeanne d'Arc）— 战斗系统与整体设计调研

> Level-5 在 PSP 上 2006 年的 SRPG，**Level-5 唯一的战棋作品**。
> Famitsu 33 分金殿堂、IGN 9.0、销量 33 万套。
> 历史改编（百年战争 + 恶魔）+ 三大原创机制（Burning Aura / Unified Guard / 变身）。
> **是 SRPG 史上最被低估的杰作之一**——比 FFT 上手友好，但深度毫不逊色。

## 一、游戏概述

| 项 | 内容 |
|---|---|
| 全名 | Jeanne d'Arc（ジャンヌ・ダルク）|
| 开发商 | **Level-5**（《暗云》《勇者斗恶龙 8》《二之国》《妖怪手表》）|
| 发行 | 2006-11 日版 / 2007-08 北美版 / Sony Computer Entertainment 发行 |
| 平台 | PSP（仅）；2024 年起在 PS4/PS5 经典游戏库（PS+）|
| 子类 | 历史改编 SRPG / 中世纪奇幻 |
| 战斗规模 | 我方 6-8 vs 敌方 8-12 |
| 单战时长 | 10-25 分钟（Boss 30-45）|
| 主线时长 | 单周目 30-40h |
| 评分 | Famitsu 33（金殿堂）/ IGN 9.0 / Metacritic 87 |
| 销量 | ~33 万套（原创战棋的高水平）|

**设计目标**：用 Level-5 的"高质量制作 + 上手友好"哲学做 SRPG ——
> "FFT/火纹太硬核，新人玩不下去。
> 我们要做一款 SRPG 老玩家觉得'有深度'、新人也'打得动'的游戏。
> 历史 + 奇幻让 IP 自带辨识度。"
> 玩法哲学：**战术机制丰富 + 入坑门槛低 + 故事自带魅力**——三轨平衡。

---

## 二、核心战斗循环

```
战前准备：
  - 选择出战角色（每个都有专属人格 + 历史背景）
  - 装配技能宝石 (Skill Stones / 晶石) + 神宝石 (Godstones)
    ↓
进入战斗 → 网格地图 + 高低差 + 侧背攻击
    ↓
关键：用普攻触发 Burning Aura（光环）
  + 站位形成 Unified Guard（联防）
  + SP 满后变身（贞德等）
    ↓
胜利条件多样化：限时杀敌 / 守护盟友 / 到达指定区
```

**关键设计哲学**：**"每场战斗都有 gimmick"** —— 不像 FFT 大多数战斗是"灭敌方"，**圣女贞德每关都有专属规则**（限时、护送、达点、连击）。

## 三、关键机制详解

### 3.1 Burning Aura（燃烧光环）⭐ 核心创新

**这是圣女贞德最具识别度的发明**：

```
任意角色用普攻命中敌人 → 在敌人对侧的格子生成 Burning Aura
  ↓
站在 Burning Aura 中的角色 → 攻击伤害 +50%
  ↓
Aura 持续 1 回合后消失（被踩的话角色可携带移动）
```

**战略意义**：
- **包围敌人 = 主动制造 Aura 链** —— 队友连续打同一个敌人 → 自动形成"3 人围攻 +50% 输出"
- **BOSS 战节奏被改写** —— 集火不是"4 人轮流戳"，是"先打主攻击者 → 制造 Aura → 让爆发型角色站 Aura 上 → 一击重伤"
- **与我们的"夹击 (Pincer)"思路高度相似**——但 FFT/TO 的夹击只是"对侧友军追加打击"，圣女贞德是 **"对侧地面留下持续性强化"**——更动态

→ 这是 SRPG 史上**第一次**把"包围"做成动态地形效果，**比 FFT/TO 更前卫**。

### 3.2 Unified Guard（联合防御）

**鼓励抱团的对偶机制**：

```
角色彼此邻接（2-3 个友军靠近）
  ↓
形成 Unified Guard 光圈
  ↓
单位防御力 +20-40%（按邻接人数）
  代价：抱团易被 AOE 一锅端
```

→ **防守战的核心机制** —— 但与 Burning Aura 形成天然张力：
- Aura 要求"分散包围敌人"
- Unified Guard 要求"队友抱团"
- **玩家必须在两者之间动态切换**——节奏感非常好

### 3.3 变身系统（Transformation） + Godspeed

**贞德等核心角色**可使用神圣臂环变身：

```
积累 SP（每回合 +1）→ 满 3-4 SP 时变身
  ↓
变身后：属性大幅提升 + 解锁专属神技
  ↓
Godspeed 机制：变身状态下击杀敌人 → 立即再次行动
  ↓
连杀链可形成无限连锁 → 战局逆转
```

**Godstones（神宝石）** 决定变身形态：
- 第 1 颗：基础形态（伤害 +Godspeed）
- 后续解锁更多形态：治疗向、AOE 向、控制向…
- **每场战斗只能变身 1 次**——必须卡时机

→ **这是 SRPG 史上的"必杀技 + 暴走模式"工业方案**：高潮戏剧性，但有严格冷却。

### 3.4 三阵营属性（Sol / Luna / Stella）

```
Sol（太阳）  克  Luna（月亮）  克  Stella（星辰）  克  Sol
```

- **石头剪刀布**式克制关系
- 每个角色基础属性偏向某一阵营
- 装备神宝石可强化阵营亲和度
- **战略层面**：根据敌方阵营偏向调整出战阵容

→ 比"无属性单 HP 池"多一层维度，比"双护甲"轻量很多。

### 3.5 Skill Gems / Skill Binding / Skill Affinities

**技能晶石系统**——比 FFT 的 Job 系统更轻量：

| 系统 | 作用 |
|---|---|
| **Skill Gems**（晶石）| 武器/装备的技能槽位，玩家自由插拔 |
| **Skill Binding**（合成）| 两块低级晶石可合成更强晶石（合成对象是《银河游侠》中的青蛙——彩蛋）|
| **Skill Affinities**（亲和）| 角色擅长某类晶石，装备时获得加成 |

**晶石分类**：
- 红色：近战主动技能
- 紫色：被动技能
- 绿色：辅助主动
- 海色：基础属性 + 阵营亲和

**装备槽随等级开放**：3 → 4 → 5 → 6（Lv30）

→ 与 FFT Job 对比：**轻量 30%、新人友好 200%、深度保留 80%**。

### 3.6 角色养成（无随机成长）

**关键差异**：圣女贞德**没有 RNG 升级**——成长完全确定：
- 升级数值固定，专心刷经验即可
- **未出战角色也获得经验**（避免冷板凳）
- 角色阵亡只本关撤退，**下关自动复活**——降低永久死亡焦虑

→ 与火纹（永久死亡）/ FFT（升级 RNG）形成鲜明对比，**新人友好 + 但有取舍**。

### 3.7 关卡设计（每关都有 gimmick）

**从未有 SRPG 把"关卡多样性"做到这种程度**：

| 关卡类型 | 例子 |
|---|---|
| 限时击杀 | "5 回合内击败 BOSS"|
| 护送任务 | "保护盟友 NPC 到达指定点"|
| 达点任务 | "在限定回合到达地图特定区域" |
| 全灭 | 经典消耗战 |
| **特殊机关** | 盗贼可架云梯绕过城门 / 隐藏宝物点 / 地形机关 |

→ 评论原话："**没有两关是一样的玩法体验**" —— 与 FFT 大多数"灭光所有敌人"形成强烈对比。

### 3.8 剧情：百年战争 + 恶魔同盟

**历史框架**：
- 1429 年法国，**圣女贞德**真实出场
- 多雷米村庄遭英军袭击，贞德获得**神圣臂环**
- 英军摄政王与**恶魔同盟**——亨利六世被恶魔附身

**剧情亮点**：
- **真实历史人物**作为可用角色：La Hire（狮形 therion）、Gilles de Rais、Charles VII…
- **大胆魔改**：贞德不被烧死，而是变身光辉骑士迎战魔王
- **20+ 段精良 anime cutscene** —— PSP 时代顶配
- **结局收尾完整**——不留续作钩子（这正是没续作的原因之一）

### 3.9 与 FFT 的明显对比

| 维度 | FFT | **圣女贞德** |
|---|---|---|
| 永久死亡 | 是 | **否（本关撤退）**|
| 升级 RNG | 是 | **否（固定成长）**|
| 通用兵种 | 多（38 职业 + 雇佣兵）| **0（每个角色都独立人格）**|
| 战斗目标 | 90% 灭敌方 | **每关 gimmick** |
| Job 系统 | 极深（38）| 轻量（晶石）|
| 上手难度 | 极陡 | **缓**|
| 玩家原话 | "深度天花板" | "**Level-5 风格的 FFT**" |

## 四、为什么好玩（抽象层）

| 体验驱动力 | 具体由什么产生 |
|---|---|
| **机制丰富但不杂乱** | Burning Aura + Unified Guard + 变身 + 阵营互克，每条都有清晰场景 |
| **关卡多样性** | "每关都有 gimmick" 是 SRPG 史上少见的产量 |
| **历史 + 奇幻的甜点位** | 真实人物 + 恶魔魔改 = 既有代入感又有戏剧性 |
| **新手不被劝退** | 无永久死亡 + 固定成长 + 撤退式失败 |
| **Level-5 制作品质** | anime cutscene + 配音（坂本真绫 / 能登麻美子 / 石田彰 / 平田广明）+ 美术 |
| **变身的高潮感** | Godspeed 连锁 = SRPG 史上最爽的"必杀时刻"之一 |

> 玩家原话（PSNProfiles）：
> "这是 PSP 上**少数你会通关后立刻开二周目**的 SRPG。"

## 五、设计缺陷与争议

### 5.1 难度尖峰（最大问题）
- 整体流畅度好，但**后半段有 2-3 场 BOSS 战难度断层**
- 评测原话："**几乎丧失通关动力**"
- 解决方案：自由刷级（地图大部分区域支持反复战斗）
- 但作者反馈："不刷的玩家会撞墙；刷的玩家觉得拖沓"

### 5.2 缺少 QoL
- **无加速 / 无跳过战斗动画**——15 分钟战斗里 3-5 分钟在看动画
- **无敌人技能完整范围预览**——只能看移动+普攻范围，看不到他们魔法/技能的真实威胁
- **无多个敌人范围同时显示**——FE 系玩家觉得倒退

### 5.3 PSP 控制方式陈旧
- 方向键控制光标 → 移动慢
- PS5/PS4 重制版未改进
- 大屏幕下 UI 像素感强，**不适合电视游玩**

### 5.4 没有续作
- IGN 9.0 + 33 万销量 → 商业上"成功但不爆"
- Level-5 转向《二之国》《妖怪手表》—— SRPG 不再做
- **国产同类作品《圣女战旗》(2019) 受其影响明显**

### 5.5 多周目内容偏少
- 二周目继承装备 + 解锁高难
- 但**故事线无分支**——没有 TO/DOS2 那种"路线选择"
- 大约 50% 的 SRPG 玩家不会开二周目

### 5.6 战斗外深度不足
- **没有装备制造 / 锻造 / 烹饪** —— 完全战斗中心
- **没有据点经营 / 公会**——单线叙事
- 与 FFT/TO 比，**世界观厚度差一截**

### 5.7 战斗中无法暂停查看
- 队友的回合开始时无法预先查看下一步敌人范围
- 必须凭记忆判断
- "硬核但反人类"

### 5.8 阵营属性偶尔失衡
- Sol/Luna/Stella 三相克 → 后期某些 BOSS 单一阵营
- 玩家被迫"出战阵容偏置"，**违背均衡队伍设计**

## 六、对我们项目的启发

### 6.1 直接拿（哲学层）
- ✅ **"每关都有 gimmick"是 SRPG 关卡设计的金标准**
  - 我们的多目标战斗（杀首领 / 护送 / 守住）已经在这条路上
  - **进一步**：每关增加一个"局部规则"——例如"BOSS 倒地后 3 回合内必须杀完小兵"
- ✅ **新手友好但保留深度的 SRPG 是真实可行的**——圣女贞德证明
  - 我们的 BB 路线偏硬核，但 UI 渐进式（关卡类型驱动）可以兼顾两端
- ✅ **历史 / 文化 IP 自带魅力**——百年战争 + 贞德 = 玩家自带情感连接
  - 中世纪雇佣兵 IP 同样自带魅力，但**要找到我们的"贞德时刻"**
- ✅ **Level-5 的"上手友好 = 减少永久死亡 + 固定成长"是工业方案**
  - 我们的"无传统等级"（参考逸剑）在这一脉上

### 6.2 直接拿（机制层）⭐⭐
- ✅ **Burning Aura（动态地形效果）—— 强烈推荐借鉴**
  - 我们的"夹击 (Pincer)"是单次追加伤害；可以**升级**为"对侧地面留 1 回合 +50% Aura"
  - 这让"包围"成为**持续性战术资产**——夹击不只是一次性触发
- ✅ **Unified Guard（站位防御加成）**——给我们"抱团 vs 包围"的经典张力
  - 实现成本低：在 Stats 加 `unified_guard_bonus`，每个相邻友军 +X
- 🟡 **变身系统（终结技 / 必杀）**—— Phase 3+ 候选
  - 给关键角色（雇佣兵首领 / 故事 NPC）准备"绝境必杀"，限制每场 1 次
- ✅ **晶石/技能合成**（Skill Binding）—— 比"装备词条"更有玩家参与感
  - Phase 3+ 起做"装备/武器升级"系统时考虑
- ✅ **关卡 gimmick 设计**——每关有 1-2 条专属规则
  - "限时杀首领 5 回合"、"NPC 必须存活"、"地形机关每 3 回合触发"
- 🟡 **三阵营属性互克**——可以做但要谨慎
  - 在我们"硬核数值真实"的赛道上引入"魔法属性克制"会出戏
  - **不推荐照搬**，但**武器克制 (穿刺克板甲、钝击克锁子甲)** 可以借鉴这条思路

### 6.3 警示（避开它的坑）
- ❌ **难度尖峰 + 强制刷级**——必须有难度自适应或多档难度
- ❌ **无加速 / 无跳过动画**——所有动画必须可中断
- ❌ **无敌人技能范围预览**——我们的"借机攻击预览"已经在做相关，要继续推
- ❌ **无续作的商业失败模式**——33 万销量在 2007 年算"成功但不爆"
  - 现代独立游戏 5 万销量就回本，不必担心
- ❌ **战斗中心化（无外围内容）**——现代玩家要"营地 / 据点 / 角色互动"
  - 我们 Phase 3+ 的"佣兵团据点"应避免这个坑

### 6.4 真正值得借鉴的"圣女贞德精神" ⭐⭐
1. **机制是为情景服务的**—— Burning Aura 不只是"伤害 +"，是"包围带来的奖励"。**任何机制都要有"叙事意义"**
2. **关卡多样性 > 关卡数量**—— 30 关每关都不同 > 60 关都是"灭敌方"
3. **新手友好不等于浅** —— 砍永久死亡 + 砍 RNG 升级，**深度反而靠机制丰富 + 关卡设计补回来**
4. **历史 IP 是免费的世界观补丁**—— 玩家先天对"贞德 / 百年战争"有情感，开发者不用花 100h 建世界观
5. **Burning Aura 的核心智慧** —— 把"被动战术（包围）"做成"主动玩法资源（地面 buff）"
   - **我们的项目最值得抄这一条**——把夹击从"瞬发追加"升级为"持续地面 buff"
6. **Unified Guard + Burning Aura 的张力** —— 一个奖励聚合、一个奖励分散，**玩家必须动态切换**——这是节奏感的源泉

## 七、结论：为什么圣女贞德是"被低估的杰作"

| 评价维度 | 状态 |
|---|---|
| 玩法创新 | ✅ Burning Aura + Unified Guard 是 SRPG 史上的发明 |
| 关卡多样性 | ✅ 每关 gimmick，少有 SRPG 做到 |
| 上手难度 | ✅ 新手友好但深度不输 FFT |
| 制作品质 | ✅ Level-5 全力（anime cutscene + 顶级配音）|
| 历史 IP | ✅ 自带魅力 + 大胆魔改 |
| 平衡 | ⚠️ 难度尖峰 |
| QoL | ⚠️ 无加速 / 无范围预览 |
| 续作 | ❌ 没有，Level-5 转型 |

**总评**：圣女贞德是 **"如果 FFT 由 Level-5 重做"** 的样子——
- 砍掉了 FFT 的硬核痛点（永久死亡 / RNG / 38 职业）
- 加入了 SRPG 史上最具创新的 3 个机制（Burning Aura / Unified Guard / Godspeed 变身）
- 用"百年战争 + 恶魔"的强 IP 撑起世界观
- **是新人开始接触 SRPG 的最佳门户作品之一**

→ 对我们项目的核心价值：
**"机制创新 + 关卡 gimmick + 历史 IP"** 这三件事，圣女贞德给了我们**全套工业方案**。
