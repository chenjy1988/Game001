# 三角战略（Triangle Strategy）—— 全技能设计与效果研究

> **来源**: Square Enix / Artdink，2022 年 3 月发售。
> **角色数**: 30 名可操控角色。
> **技能模型**: 三段职业 × 三段武器 = 双轨并行成长。

---

## 一、技能系统架构

### 1.1 双轨并行

```
角色升职（3 段职业）            武器强化（3 段等级）
  消耗勋章                       消耗素材
  └→ TP 上限 +1                  └→ 解锁新技能
  └→ 解锁技能槽                  └→ 解锁武器奥义（每人 1 个）
  └→ 属性成长                    └→ 降低 TP 消耗
```

### 1.2 技能类型（每角色 5~8 个）

| 类型 | 人均数量 | TP 消耗 | 说明 |
|---|---|---|---|
| 主动技能 | 3~4 | 1~4 TP | 菜单选择发动 |
| 被动技能 | 1~3 | 无 | 永久生效 |
| 武器奥义 | 1 | 2~5 TP | 武器满级解锁，每人专属 |

### 1.3 TP 机制

| 阶段 | TP 上限 | 每回合自然回复 |
|---|---|---|
| 基础职业 | 3 | +1 |
| 进阶职业 | 4 | +1 |
| 终极职业 | 5 | +1 |

## 三、技能设计规律总结

### 3.1 技能分布模式

| 角色类型 | 主动技能 | 被动技能 | 奥义效果倾向 |
|---|---|---|---|
| 物理近战 | 3~4 | 2~3 | 单体高伤 / 无视防御 |
| 法师 | 4~5 | 2~3 | 范围 + 地面点燃 + 全消耗爆发 |
| 辅助/Buff | 4~6 | 1~2 | 全体无敌 / 双动赋予 |
| 坦克 | 3~5 | 2 | 全体免疫 / 替身承伤 |
| 远程控制 | 4~6 | 1 | 范围 Debuff |
| 治疗 | 4~6 | 1 | 范围持续回复 |

### 3.2 TP 消耗节奏

```
1 TP：小技能（buff / 护盾 / 延迟斩），可频繁使用
2 TP：主力技能（单体高伤 / 中范围），核心战术
3 TP：高级技能（冲锋 / 穿透 / 再动赋予），需要积蓄
4~5 TP：奥义，第 3~4 回合就绪
```

### 3.3 角色差异化手段

- **独有机制**：安娜双行动、本尼迪克调序、艾拉德替身、吉拉复活
- **地形联动**：芙蕾德丽卡点燃地面 + 站在火上增伤
- **天气触发**：罗兰晴天移动+1、芙蕾德丽卡晴天火伤↑
- **条件触发**：罗兰移动后追击、芙蕾德丽卡击杀回 TP
