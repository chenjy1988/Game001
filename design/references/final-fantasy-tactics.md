# Final Fantasy Tactics — 战斗系统与整体设计调研

> 松野泰己 1997 年的 SRPG 神作；2007 年 PSP《War of the Lions》、2025 年《The Ivalice Chronicles》是其当代化身。
> 是 CT 行动条 + 政治叙事的双重源头，**TO 的精神续作**——同一编剧、同一世界观哲学、更激进的战斗系统。

## 一、游戏概述

| 项 | 内容 |
|---|---|
| 全名 | Final Fantasy Tactics（FFT，2025 重制版 The Ivalice Chronicles）|
| 创作者 | 松野泰己（编剧/导演，原 Quest 团队，TO 同一作者）|
| 开发商 | Square（原版）/ Square Enix（重制）|
| 作曲 | 崎元仁（Hitoshi Sakimoto）+ 岩田匡治（Masaharu Iwata）|
| 发行 | 1997 PS1 / 2007 PSP《狮子战争》/ 2025《伊瓦利斯编年史》|
| 平台 | Switch、PS4/5、Xbox、PC（Ivalice Chronicles）|
| 子类 | 等距视角格子战棋 / 政治剧情 SRPG |
| 战斗规模 | 我方 5 人 vs 敌方 5–10 |
| 单战时长 | 15–40 分钟（Boss 60+）|
| 主线时长 | 单周目 50–80h；全 Job 解锁 80h+ |
| 宏观循环 | 章节叙事 → 关键选择 → 战斗 → Job 解锁 → 下一章 |

**设计目标**：用 SRPG 玩法承载"封建战争中的阶级寓言"——
> "战争中没有英雄。受压迫的平民拉姆萨与被夺走童年的迪利塔，
> 一个走上对抗腐朽，一个走上篡权称王。
> 教会的圣战只是阶级阴谋的伪装。"
> 玩法哲学：**Job 自由组合 + CT 行动条 = 让战术创造力替代刷级**。

---

## 二、核心战斗循环

```
战前部署（最多 5 名单位）
    ↓
按 CT 行动条排序，单位轮流行动
    ↓
每次行动：移动 + Act（攻击/技能/咏唱）+ Wait
    ↓
胜利条件：击杀 Leader / 全灭 / 守住 / 护送
    ↓
战利品 + Job 经验 + 角色经验
```

**关键设计哲学**：FFT 是 **CT 行动条的鼻祖**——
- 没有"我方回合 / 敌方回合"
- 没有 AP 池（这是与战兄弟最大差异）
- 战术深度来自 **Job 自由组合 × 行动条节奏 × 地形高度**

## 三、关键机制详解

### 3.1 CT (Charge Time) 行动条系统（核心）

**工业级 CT 实现**——所有 CT 派系游戏的源头：

```
每个 Tick（时钟节拍）：
  for unit in alive_units:
    unit.ct += unit.speed

  if any unit.ct >= 100:
    Tick 暂停，CT 最高的单位行动
```

#### Speed 公式
- Speed 是**唯一**决定行动频率的属性
- 常见单位 Speed 5–12，加上装备/Buff 可到 15+
- **Speed +1 ≈ 行动频率 +10%**（在 8–12 区间内）
- "Speed Save"被动（被攻击时几率 +1 Speed）能在长战中累积压倒性优势

#### 行动后 CT 扣除（关键平衡装置）

| 行动 | CT 扣除 | 含义 |
|---|---|---|
| **Act + Move**（行动+移动）| 100 → 0 | 完整回合 |
| **仅 Act 或仅 Move** | 80（保留 20）| 没用全资源，下次更快 |
| **Wait（等待）** | **60（保留 40）**| 不动不打，**下次提前一轮** |
| 行动被回避/打断 | 80 | |

→ **这就是 TO Reborn 设计的"WT cost"原型**——FFT **早 25 年**就用这个机制平衡了高速单位。

#### Wait 是核心战术资源
- "Wait" 不是消极动作，是**主动出招**
- 用途：
  - 调整出手顺序（让伙伴先放范围 buff 再行动）
  - 等 Reaction Ability 触发位置
  - 避免站到敌人 AOE 中

→ FFT 玩家的**"打高难"几乎完全依赖 Wait 节奏**——比 TO Reborn 还硬核。

### 3.2 咏唱（Charge）机制 —— 第二个 CT

**FFT 的隐藏天才设计**：魔法/召唤/必杀技用**独立的咏唱 CT**，与主 CT 解耦。

```
玩家选择 "Fire 2"（Charge=15）
  → 主 CT 归零，进入"咏唱中"状态
  → 咏唱 CT += Speed（每 Tick）
  → 咏唱 CT >= 15 时，魔法生效
  → 期间不能动、不能格挡、易被打断（损 Faith / 受伤）
```

→ **"Speed=8 / Fire 2 Charge=15"** = 约 2 Tick 完成 = **快咏唱有质变**。

### 3.3 Haste / Slow / Stop —— CT 修正状态

| 状态 | CT 累加倍率 | 战术意义 |
|---|---|---|
| **Haste** | × 1.5 | 行动频率 +50%；同样加速咏唱 |
| **Slow** | × 0.5 | 行动频率 -50% |
| **Stop** | × 0 | CT 完全停滞，敌人无法行动 |
| **Sleep / Petrify** | × 0 | 同上但额外效果 |

→ Slow + Stop + Don't Act 是 **FFT 锁 BOSS 的最强组合**。
→ 这套**直接修改 CT 累加速度**的设计，比 BB 的"疲劳间接降 init"更激进、更直接。

### 3.4 Job（职业）系统 —— SRPG 史上最深的之一

#### 38+ 职业 / 自由换职 / 双职业槽位

- **38 个通用职业 + 11 个专属职业** + 怪物/客串单位
- 任何单位可在战斗外**自由切换**至已解锁的职业
- 切换职业**不会损失等级**（与 BB 的 background 系统差异巨大）
- 每个单位有 **2 个职业槽**：
  - **主职业**（Primary）：决定数值成长、可装备武器
  - **副技能**（Secondary）：从其他职业借用一套技能

#### Job 解锁链（深度核心）

```
Squire (新手) → 解锁 Knight + Archer + Chemist
  Knight → 解锁 Monk
    Monk + Lancer → 解锁 Geomancer
      Geomancer + Wizard → 解锁 Time Mage
        Time Mage 4 → 解锁 Arithmetician
        ...
```

→ **解锁高级职业要花 30+ 小时**——但这正是 FFT 的"长期目标驱动力"。

#### Job 组合的爆炸力

| 组合 | 效果 |
|---|---|
| Knight 主 + Time Mage 副 | 物理战士能瞬移 / 减速 |
| Monk 主 + Calculator 副 | 拳法 + 全屏数学魔法 |
| Ninja + 任何远程 | 双持武器，每回合输出双倍 |
| Mime（模仿者）+ 强力角色站队 | 0 cost 复制队友的强力技能 |

→ 评测人称："**FFT 是 theorycrafter 的天堂**——你能在第 2 章就 break the game"。

### 3.5 政治叙事 —— SRPG 史上最尖锐的剧本

#### 故事框架
- **舞台**：伊瓦利斯（Ivalice），五十年战争结束后的封建王国
- **主角**：拉姆萨（Ramza）—— 贵族 Beoulve 家族幼子
- **挚友**：迪利塔（Delita）—— 平民出身，后篡位成王
- **背景势力**：Glabados 教会 —— 用宗教伪装的政治阴谋

#### 主题深度（远超同类 SRPG）

| 主题 | 在游戏里怎么呈现 |
|---|---|
| **特权的盲区** | 拉姆萨开场围剿"因家人挨饿沦为盗贼"的退伍军人 |
| **阶级战争** | 富人把穷人视为非人，平民被战争碾碎 |
| **宗教武器化** | 教会煽动种族仇恨、妖魔化"异教徒" |
| **抵抗的必要性** | 拉姆萨从顺从到反抗的成长 |

> 评测原话（Rolling Stone, 2025）：
> "FFT 不是奇幻冒险——是用奇幻外壳的政治寓言。
> 灵感来自日本泡沫经济崩溃，'希望被价格标签衡量'的时代。"

#### 与 TO 的关系
- 同一编剧（松野泰己）、同一作曲（崎元仁/岩田匡治）、同一美学
- TO 早 2 年（1995），但 FFT **更激进**——TO 的 Law/Chaos 是路线选择，FFT **直接讲阶级革命**
- 评测人称 FFT 为 "TO 的精神续作"，但**剧本更尖锐**

### 3.6 五十年战争世界观深度

- **State of the Realm 系统**（2025 新增）：玩家可随时查阅当前剧情进度下各国势力关系
- **Codex（百科）**：每个角色、地点、事件、神祇都有详细背景
- **Brave Story**（PSP/Ivalice 新增支线集）：不影响主线但补充世界观
- **Voice Novels**（声音小说）：4 部原日版被英文版砍掉的内容，2025 年回归

→ FFT 的世界观**比 TO 还要丰富**——可惜原版 PS1 时代很多内容没本地化。

### 3.7 战斗界面与 QoL 改进（2025 重制版）

| 项 | 原版 PS1 | 2025 Ivalice Chronicles |
|---|---|---|
| 行动顺序预览 | ❌ | ✅ 侧边栏显示未来 N 步 |
| 撤回已确认动作 | ❌ | ✅ 可撤回移动 |
| 难度切换 | ❌ | ✅ 三档可随时切换 |
| 战斗加速 | 1x | 2x（无 3x，仍被吐槽）|
| 全语音 | ❌ | ✅ 英日双轨 |
| Codex/百科 | ❌ | ✅ 移植自 FF16 |
| 战斗中对话 | ❌ | ✅ 剧情角色会插话 |
| 撤销操作 | ❌ | ✅ 移动可撤销 |

### 3.8 地形高度系统

- **每个 hex 有独立高度数据**——FFT 是首批做高度的 SRPG
- 高度差影响：
  - 命中率（高位攻击 +）
  - 伤害（高地攻击伤害加成）
  - 远程射程（射高低分别有惩罚 / 加成）
  - 移动消耗（爬坡耗 CT 更多）
- **跳跃力（Jump）**作为单独属性，决定能跨越多高的高度

→ 这比"平地战棋"多一个完整维度，是**TO 也不及 FFT 的强项**。

## 四、为什么好玩（抽象层）

| 体验驱动力 | 具体由什么产生 |
|---|---|
| **CT 行动条节奏感** | 行动条让玩家"看见时间"；Wait 让玩家"操控时间" |
| **Job 自由组合** | 38 职业 × 2 槽 = 数百种 build；玩家创造力替代刷级 |
| **政治剧本的深刻** | 松野编剧 + 阶级寓言 + 宗教阴谋 |
| **战斗即叙事** | 每场战斗都有对白 / 立绘 / 剧情张力，不是"消耗体力" |
| **theorycrafting 天堂** | 早期就能 break the game，但仍有挑战空间 |
| **音乐的颗粒度** | 每张地图都有专属配乐，崎元/岩田双 BGM 大师 |

> 评测原话：
> "战斗与剧情**毫无疑问是顶级的** —— 但糟糕的 QoL 严重削弱了体验。"

## 五、设计缺陷与争议

### 5.1 QoL 灾难（2/10 评分项）—— 最严重的问题
- **强制随机遇敌**：单战可达 40 分钟，但仍触发随机战，"几乎让我放弃游戏"（评测原话）
- **无三倍速**：2x 太慢
- **战斗动画无法跳过**（部分召唤兽 30 秒 cutscene）
- **菜单层级深**：转职、装备、技能要点 5-6 层菜单

### 5.2 平衡性灾难
- **职业天差地别**：Calculator / Mime / Cid 是"破坏游戏"级别
- **黑魔会误伤队友**——AOE 没敌我识别
- **Mime 模仿能力使用方式诡异**——必须站 4 格内
- 玩家原话："**花 5 小时解锁 Lich，发现是垃圾**"

### 5.3 难度跳跃极大
- 第 1 章简单到无聊
- 第 3 章 **Wiegraf 单挑** 是劝退点（无解封锁玩家进度）
- 后期支线（Deep Dungeon、Orbonne）难度断层

### 5.4 grind 是必修课
- 解锁每个新 Job 都要刷 Job Points
- 后期 4 名客串角色全要刷支线
- BB 玩家对此适应良好；FFT 新手会被刷哭

### 5.5 场景复用严重
- **没有完整世界地图**（只有点位 + 路径）
- 战斗地图大量复用
- 评测打 5/10

### 5.6 支线全在后期
- Brave Story、Cloud、Balthier、Luso、Deep Dungeon 全堆在 Ch4
- "经历大量随机战斗后已感到疲惫，再做支线没动力"

### 5.7 角色塑造不均
- Ramza / Delita / Wiegraf / Algus / Agrias 出色
- 其余通用单位**没人格、纯机制**——没人会爱上 Squire #3
- 缺乏伙伴情谊——Ramza 没有"非剧情角色的真朋友"

### 5.8 美术与音乐时代隔阂
- 2D 像素 + 等距，1997 标准下顶级，**2025 看起来年代感强**
- 2025 重制只 remaster 没 HD-2D，与三角战略对比落后
- 音乐评 7/10，作曲家是 Sakimoto/Iwata 但"循环略显重复"

## 六、对我们项目的启发

### 6.1 直接拿（哲学层）
- ✅ **CT 行动条 + Wait 动作**——FFT 的"Wait 扣 60 而非 100"是高速单位平衡的工业方案，比加 fatigue 直观
- ✅ **战斗即叙事**——每场战斗给点对白、立绘、目标多样化，避免"清场感"
- ✅ **Job 自由换职**（如果未来要做 background 系统的话）——保留等级、低成本试错
- ✅ **行动顺序预览 UI**（已经在做）——FFT 2025 做侧边栏 8 步预览，与我们方向一致
- ✅ **战斗中对白**——剧情角色出战时插话，提升沉浸度

### 6.2 直接拿（机制层）
- ✅ **Wait 动作**—— FFT 的隐藏神来之笔。给玩家一个"等待 60 CT 而不是 100"的选项，立刻打开战术深度（避 AOE / 调出手序 / 等队友 buff）
  - 实现：在 BattleScene 加 `Wait` 按钮（空格键已经是 end_turn，可以保留语义）
  - 数值：扣 60 CT 而不是 100
- ✅ **咏唱机制**——给"重击 / 蓄力技能"做"主 CT 归零 + 独立咏唱 CT"双轨。让玩家可以打断 / 提前出招避开
- ✅ **Haste/Slow 状态**——直接修改 CT 累加倍率，比"疲劳间接降 init"清晰
- 🟡 **地形高度**（已在 Phase 2 待办里）——FFT 证明这是一个完整维度，值得做

### 6.3 警示（避开它的坑）
- ❌ **强制随机遇敌**——已经避开（无随机遇敌）
- ❌ **38 职业**——复杂度爆炸，我们走"武器关键字 + 装备组合"路线
- ❌ **AOE 误伤队友**——可以保留作为高级技能的特性，但默认要有"不误伤"开关
- ❌ **必杀技/召唤 30 秒 cutscene**——所有动画必须可跳过
- ❌ **支线全堆后期**——Phase 3 起设计任务时要分散
- ❌ **职业平衡灾难**——5-8 个原型而非 38，便于精调

### 6.4 真正值得借鉴的"FFT 精神"
1. **Wait 是核心战术，不是放弃** —— "扣 60 而非 100"这一条规则**单独打开了一整层深度**
2. **CT 行动条是对玩家"叙事节奏"的承诺**——不是"快单位多动"，是"玩家看见时间流"
3. **政治叙事要尖锐** —— FFT 让"贵族-平民"成为玩法张力的一部分（拉姆萨的家族决定了一些战斗的目标）
4. **theorycrafting 是 SRPG 的长期粘性** —— 玩家会回来"试新组合"，不是"通关一次"
5. **战斗中对白** —— 比"过场动画"便宜，比"沉默战斗"有趣 100 倍
