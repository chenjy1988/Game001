# XCOM (Firaxis 2012-2016) — 战斗系统设计调研

> "战术 SRPG 的现代化模板"——把 SRPG 从 35+ 老饕重新打开给主流玩家。
> 本文聚焦 Firaxis 重启系列（XCOM: EU/EW + XCOM 2 + WOTC），
> 必要处对比 1994 年原版 X-COM: UFO Defense。

## 一、游戏概述

| 项 | 内容 |
|---|---|
| 系列重启者 | Firaxis Games（席德梅尔工作室）|
| 主创 | Jake Solomon（创意总监）|
| 关键作品 | XCOM: Enemy Unknown (2012) → Enemy Within (2013) → XCOM 2 (2016) → War of the Chosen (2017) |
| 平台 | 全平台（PC / Console / Mobile）|
| 子类 | 现代战术 / 大众化 SRPG |
| 战斗规模 | 通常 4–6 vs 6–12 |
| 单战时长 | 15–25 分钟（XCOM 2）|
| 宏观循环 | 全球战略 → 部署 → 战术战斗 → 资源/科技 → 长休 |

**设计目标**：让"硬核 SRPG 体验"在**主流玩家手中可消化**。
- 决策必须**直觉**（两动作点制）
- 失败必须**有意义但不毁档**（永久死亡 + 替补深度）
- 信息必须**透明**（命中率/掩体类型一目了然）
- 节奏必须**紧凑**（倒计时回合 + 短战斗）

## 二、核心战斗循环

```
战略层选择任务（含 Avenger 飞船管理）
    ↓
战术层：4–6 名士兵部署（XCOM 2 含隐蔽阶段）
    ↓
回合开始 → 每名士兵 2 个行动点
    ↓
首次发现敌方 → Pod 激活动画 → 敌方免费移动到掩体
    ↓
玩家 vs 敌方交替整回合
    ↓
（任务可能含倒计时）N 回合内达成目标
    ↓
撤离 → 长休/伤员/战利品/晋升
```

**两动作制（Two Action System）**——这是 XCOM 重启系列的灵魂：

| 行动组合 | 效果 |
|---|---|
| 移动 + 攻击 | 标准 |
| 移动 + 移动（Dash）| 双倍移动距离 |
| 攻击 + 攻击 | 仅特定职业 / 技能 |
| 移动 + Overwatch | 准备阶段反应射击 |
| 不动 + 瞄准 | 部分职业触发命中加成 |

> 老 X-COM 用 TU（Time Units，65–80 时间点）做精细调度，
> 重启版砍成"2 个行动点"是**最大的简化决断**。

## 三、关键机制详解

### 3.1 两动作系统（Two Actions Per Turn）

每个士兵每回合：
- 第一个行动可以是**任何**（移动、攻击、技能）
- 第二个行动**只能是**：移动、攻击、Overwatch、Reload、特定技能
- "攻击后不能移动"——这是核心约束

**意义**：
- 把战术决策**压缩到二选一 / 三选一**
- 玩家不需要心算 65 TU 怎么分配，只需问"这名士兵要打 + 走 还是 走 + 守"
- **学习曲线极平滑**——5 分钟教程后玩家就能上手

### 3.2 Pod 激活机制（Pod Activation）

XCOM 系列的争议设计：
- 敌方以 **2–4 人小组（Pod）** 散布在地图上
- 第一次进入 XCOM 视野时，触发 **激活动画**
- 动画期间敌方**自由移动到掩体**（不消耗它的回合）
- 动画结束后，进入正常回合制

**意义（设计意图）**：
- 创造"突袭 / 被突袭"的戏剧性时刻
- 让"侦察"成为有意义的行动——而不是"反正所有敌人一开始就上"
- 模拟现代战术的"接战不可预测性"

**意义（玩家心智）**：
- 主流玩家觉得"刺激"
- 硬核玩家觉得"反智"——为什么我看见敌人，敌人能免费跑？
- 后来 XCOM 2 加入**隐蔽（Concealment）**机制部分修复了这个问题（见 3.3）

### 3.3 隐蔽阶段（Concealment，仅 XCOM 2）

- 大部分任务开局，XCOM 小队处于"隐蔽"状态
- 敌方不会主动发现你，除非你进入它的"侦察锥"或开火
- 玩家可以**摆好阵型再发起伏击**——伏击的第一击有命中加成且免费

**意义**：
- 把"Pod 激活"的劣势**反过来交给玩家**——你也能伏击 AI
- 让玩家在战斗开始前**已经赢了一半**
- 是 XCOM 2 相比一代最大的体验升级

### 3.4 命中率体系 + 掩体

公式（简化）：
```
基础命中率（武器 / 距离 / 技能）
- 目标掩体修正（半掩体 -20，全掩体 -40）
+ 高低差（高位攻击 +20）
+ 侧翼（绕到掩体侧面 = 无掩体 + 暴击概率）
+ 瞄准技能（Hunker Down 反向：自己进入更高掩体）
- 烟雾 / 视线遮挡
```

**透明性**：所有修正显式显示，鼠标悬停可见每一项。

**掩体的革命性**：
- 把"二维方格"升维成"二维 + 掩体方向"
- **侧翼（Flanking）**成为核心战术——绕到背后无掩体且 +暴击
- 玩家被迫**关心地图地形**而不是只看格子距离

### 3.5 倒计时任务（Mission Timers）

XCOM 2 的标志性设计：
- 大多数任务带 **N 回合倒计时**（典型 8–12 回合）
- 超时 = 任务失败 / 部分失败
- 与隐蔽阶段配合：你**可以**慢慢摆阵型，但**不能**永远摆

**意义**：
- 把"龟缩流"——XCOM EU 的最优解——直接打死
- 强迫玩家**前压**而不是固守掩体被动反应
- 让"激进 vs 保守"成为**真实的资源决策**而不是"反正等敌人来"

**争议**：
- 玩家投票"最讨厌的设计"长期榜首
- 部分硬核 mod（Long War 2）调整或删除倒计时
- Firaxis 在 WOTC DLC 里给了部分任务"倒计时延后到接战才开始"的妥协

### 3.6 同步随机数（Synchronous Random）

> Jake Solomon 原话:
> "We use synchronous random in combat so the player can't just reload when they miss a shot."

- 命中率 RNG **种子在保存时一起存**
- 玩家 SL 后开同样的枪，**仍然命中或仍然未中**
- 想躲掉 miss，必须**做不同的决策**（换士兵 / 换目标 / 换技能）

**意义**：
- 把"SL 大法"在游戏哲学层面打死
- 强迫玩家**接受失败 + 适应**——这是 XCOM 系列"叙事自发性"的根基
- 让"关键 miss"成为故事的一部分而不是 bug

### 3.7 难度感知 RNG（低难度的"善意作弊"）

- **Normal 难度**：玩家显示 80% 命中率时，**实际**约 88%（隐藏加成）
- **Classic 及以上**：完全公平 RNG
- **Impossible**：部分情况下 AI 也获得轻微命中加成

**意义**：
- "命中率心理学"——人类大脑感受到的命中概率比实际值低 7–10%
- 低难度修正**对应玩家直觉**而非数学
- 这是"对玩家友善"的工程化实现

**争议**：
- 部分玩家觉得被欺骗（"为什么显示 80% 实际是 88%？"）
- 但 Solomon 坚持："比起一致性，玩家直觉更重要"

### 3.8 永久死亡 + 替补深度

- 士兵死亡**永久**——没有复活道具
- 但晋升体系让**替补也能快速成长**——避免一两个精英带全队
- 长休 / 伤员系统强制**多线培养**

**对比 BB 的差别**：
- BB 把"永久死亡"当作沉重叙事——玩家心痛
- XCOM 把"永久死亡"当作**战术资源管理**——你不能依赖任何一个英雄
- 体验更轻，但底层规则同等硬核

### 3.9 士兵职业 + 技能树

四大基础职业（XCOM 2）：
- **Ranger**：近战 + 隐匿（剑可一击斩敌）
- **Sharpshooter**：狙击 + 手枪（远距离 + 连射）
- **Grenadier**：爆破 + 重武器（破坏掩体）
- **Specialist**：无人机 + 黑客（治疗 / 控场）

每职业 6 级技能树，每级 **二选一**——共 64 种构筑组合。

**意义**：
- 用"职业"做**强身份**，避免 BB 那种 30+ 字段属性表的认知负担
- "二选一"逼出真正的取舍，避免"全 spec 通吃"
- 战斗中职业之间形成**显式协同**——而非数值叠加

### 3.10 战斗与战略层耦合

每次战术战斗的结果**直接影响**：
- 全球抵抗网络的进度
- 士兵的伤情 / 心理状态
- 后续任务的难度（连胜会触发 Avatar 项目反扑）
- 战利品（XCOM 2 含尸体回收）

**意义**：让"打这一仗"在**两层都有意义**——不是只在战术层。

## 四、为什么好玩（抽象层）

| 体验驱动力 | 具体由什么产生 |
|---|---|
| **决策直觉** | 两动作制 + 透明命中率 + 显式掩体 |
| **戏剧性瞬间** | Pod 激活 + 同步随机 + 永久死亡 |
| **节奏紧张** | 倒计时任务 + 隐蔽伏击 + 短战斗 |
| **学习曲线友善** | 5 分钟教程上手 + 难度感知 RNG |
| **长期投入** | 战略层耦合 + 士兵养成 + 替补深度 |
| **叙事自发性** | "我那个特等兵 Smith 在第 23 任务断了腿..." |
| **"再来一次"驱动** | 同步随机 + Ironman 模式让每次决策有重量 |

> Firaxis 把"战兄弟级别的硬核体验"翻译成了"主流玩家能消化的语言"——
> 这是 XCOM 系列在游戏史上的真正贡献。

## 五、设计缺陷与争议

### 5.1 Pod 激活的"反智"质疑
- 玩家长期诟病
- "为什么我看见你，你还能免费跑？"
- 是用"戏剧性"换"逻辑性"的设计选择，部分玩家不买单
- Long War 等 mod 提供了"激活后必须保持当前位置"的修正版本

### 5.2 倒计时任务的体验割裂
- 玩家投票"最讨厌的设计"长期第一
- 倒计时 + 永久死亡 → 部分玩家陷入"不敢前压也不能不前压"的瘫痪
- WOTC 部分修复，但根本张力仍在

### 5.3 同步随机数的玩家反弹
- 想 SL 改运气的玩家被堵死
- 部分玩家觉得"不让我读档是侮辱我"
- mod 社区有"Random RNG"补丁删除同步性
- **设计师立场**：这是哲学选择，不是 bug

### 5.4 命中率心理学的争议
- 80% 实际 88% = "你在欺骗我"
- 但**完全公平 RNG 的体感更糟**
- 这是"诚实 vs 友善"的两难，没有完美答案

### 5.5 失败的级联效应过强
- 关键节点失利 = 死亡螺旋
- 资源系统（supplies / 科技 / 工程师）一旦落后，难以追回
- 主流玩家在这个曲线下放弃

### 5.6 资源种类繁多
- supplies / 外星科技 / 科学家 / 工程师 / 电力 / 通信容量 / 空间
- 战略层 UI 比战术层重得多
- 部分玩家只想打仗，被战略层劝退

### 5.7 后期同质化（XCOM 2）
- 中后期 Magnetic Coil → Plasma 是固定升级路径
- 敌方"Sectopod / Gatekeeper"成为"必须破坏掩体后侧翼"的固定解
- 战术多样性后期下降

### 5.8 老 X-COM 玩家的反弹
- 失去 TU 精细调度
- 失去夜战 / 视野系统的复杂性
- 失去"基地防御 + 多基地"的宏观深度
- "XCOM 是简化版的 X-COM"——硬核老玩家长期不满

### 5.9 永久死亡 + 倒计时的耦合压力
- 这两个机制单独都好
- 叠加时**对新玩家过于严苛**
- 部分玩家不得不开 Easy 难度才能体验完整剧情

### 5.10 AI 的关卡脚本依赖
- 大部分"激活的 Pod"行为可预测
- 玩家学会"骗 Pod 进 Overwatch"——破坏紧张感
- AI 自身缺乏全局战略意识（mod 社区有"Better AI"补丁）

## 六、对我们项目的启发

### 6.1 直接拿（机制层）
- ✅ **透明命中率**——已实现
- 🔴 **未来回合行动条**（参考 Berwick / XCOM）——
  在战斗 UI 顶部显示未来 5 步行动顺序，让混合回合可读
- 🔴 **职业身份感 > 字段堆砌**——
  我们的武器关键字制就是这个思路的浓缩版

### 6.2 直接拿（哲学层）
- ✅ **决策直觉**——两动作制启发：减少 tooltip 阅读量
- ✅ **同步随机的 SL 哲学**——可考虑在"硬核模式"开启
- ✅ **难度感知 RNG**——
  > **强烈建议**：常规难度下做"伪随机保护"——90% 命中**保证**至少 19/20 不连 miss 两次
  > 这是 BB **没做**的、XCOM 做了的、玩家体验差异巨大的关键设计

### 6.3 选择性拿（按定位调整）
- 🟡 **倒计时任务**——
  - 不在每一关上倒计时（XCOM 2 的体验教训）
  - 但**特定关卡**用倒计时（追击 / 守护对象 / 撤离）
  - 让"压力 vs 摆阵"成为**关卡级别**的差异
- 🟡 **隐蔽 / 伏击阶段**——
  - 我们的高低差地形可以挂"侦察锥"
  - 部分关卡允许玩家先布阵再触发战斗
- 🟡 **职业固定 + 技能树二选一**——
  - 长期可能演化方向
  - 但 Phase 1 仍以"角色 + 武器"为身份核心

### 6.4 警示（避开它的坑）
- ❌ **Pod 激活动画**——
  - 我们的设计应**避免**"敌人首次出现可免费移动"
  - 直接让所有单位走速度排序，不做特殊激活
- ❌ **资源系统多到劝退**——
  - 战略层（如做）必须比 XCOM 2 简洁 50%
- ❌ **倒计时滥用**——
  - 倒计时只用在**叙事必要**的关卡
- ❌ **命中率心理学的"善意欺骗"**——
  - 显示 80% 就是 80%
  - 但**伪随机保护**可以做（连 miss 概率上限）
  - 这是"诚实 + 友善"的折衷

### 6.5 真正值得借鉴的"XCOM 精神"
1. **5 分钟教程能上手**——这是"轻快"的真实定义
2. **每个决策都直觉**——两动作 / 透明命中 / 显式掩体
3. **失败有故事**——同步随机让"我那个 Smith 中士..." 成为玩家叙事
4. **简化 ≠ 浅薄**——XCOM 把 X-COM 砍掉一半字段，深度只降 20%
