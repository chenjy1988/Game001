# Berwick Saga — 战斗系统设计调研

> 加贺昭三在《火焰纹章》之父名号下交出的"严苛但优雅"的 SRPG 答卷。
> 中文圈译名"泪之腕轮 2"（Tear Ring Saga 续作）。

## 一、游戏概述

| 项 | 内容 |
|---|---|
| 全名 | TearRing Saga Series: Berwick Saga（ベルウィックサーガ）|
| 创作者 | 加贺昭三（《火焰纹章》系列原作者）|
| 开发商 | Tirnanog |
| 发行 | 2005，Enterbrain |
| 平台 | PlayStation 2（仅日版，无官方英化）|
| 子类 | 经典 SRPG / 加贺昭三系 |
| 战斗规模 | 通常 8 vs 12 ~ 20 |
| 单战时长 | 30–60 分钟 |
| 宏观循环 | 城镇探索 → 雇佣 → 部署 → 战斗 → 战利品/赎俘 |

**设计目标**：把"战术决定胜负"做到极致。
通过**减法制公式 + 混合回合 + 真实摩擦感**，让玩家相信
> "我赢，是因为我想清楚了；我输，是因为我没想清楚——不是因为骰子坑我。"

---

## 二、核心战斗循环

```
战前部署（含选择"派去援军方向 vs 主战场"）
    ↓
混合回合开始（敌我交替，比例≠1:1）
    ↓
轮到你的某个单位 → 移动 + 攻击/技能
    ↓
血量归零 ≠ 死亡 → 进入"受伤"状态 → 若不撤退则被俘
    ↓
直到达成任务目标（胜利 ≠ 全灭敌方）
```

加贺昭三的标志性设计：**胜利条件几乎从不"杀光"**——而是"护送 X 到 Y"、"守住 N 回合"、"诱杀首领"。这一条让"打不过就跑"成为合法战术。

## 三、关键机制详解

### 3.1 减法制伤害公式（核心）

```
伤害 = 攻击力 + 武器威力 - (防御 + 盾威力)
若 ≤ 0：伤害为 0（敌人毫发无伤）
```

**意义**：
- 这是 FE 系的传统公式，但 Berwick 玩到了极致
- 防御一旦超过攻击，**常规武器完全打不动**——重甲怪在这套公式里是"墙"
- 公式简单到玩家**心算就能做战术**——不用打开计算器

**这套公式的体验代价**：
- 重甲必须有"反制工具"，否则游戏卡死
- → **直接导致了匕首特例的存在**（见 3.2）

### 3.2 匕首"无视防御"特例

匕首（Knives）的特殊条款：
1. **敌方防御视为 0**（核心点）
2. **使用者力量也不计入**——只靠武器威力本身
3. **负伤率是普通武器的 3 倍**（9% vs 3%，永久减属性）
4. 匕首本身**威力很低**（一般 5–10）

数值示例：
| 目标 | 普通剑 (Str 15 + Wpn 10) | 匕首 (Wpn 8) |
|---|---|---|
| 重甲（防御 20）| 25 − 20 = **5** | **8** |
| 轻甲（防御 5）| 25 − 5 = **20** | **8** |

**关键洞察**：Berwick 的匕首**不是"更强的武器"，而是"高甲反制工具"**。
打软目标性价比反而极差。

> 这是减法制下的**优雅破局**——不引入新维度，而是用一条特例把死局打开。

### 3.3 六边形战场

- 第一款使用六边形格的火焰纹章 / TearRing 系作品
- 包围强敌更困难——敌人多了 2 条逃跑路线
- 即使是"堵城门"这种基本任务，也因角度暴露而需要重新思考阵型

### 3.4 混合回合制（Mixed Turn）

> 不再有"我方回合 / 敌方回合"的清晰划分。

- 敌我单位以**不均匀模式**交替行动
- **人多的一方获得更多行动机会**——但不是连续 N 个
- 战术可能性：
  - 把弱单位安全放在已行动的敌人旁边
  - 优先消灭尚未行动的敌人
  - 盗贼可以在敌人围攻前提前逃脱

**意义**：这是**对 FE 系最大胆的改造**。
传统"我全动完→敌全动完"的节奏被打破，玩家无法靠"反正下回合是我的"的预设来推算。

> 评论家原话："让玩家更容易锁定优先目标、保护队友"——
> 因为你**知道哪个敌人下一动**，**而不是"敌方那 12 个的某一动"**。

### 3.5 受伤 / 缴械 / 俘虏系统

- HP 归零 ≠ 死亡 → 进入"受伤"状态
- 受伤单位需狼狈撤退（移动力大幅下降）
- 撤退失败 → 被俘 → 战后用钱赎回

**意义**：
- 软化"永久死亡"的精神压力——但保留"代价"（撤退要回合数 / 赎金要钱）
- 让"重伤员一定要救"成为**回合内的真实战术决策**——不是 Game Over 担保
- 比 FE 的"死了就死了"更**让玩家投入**，比无代价复活更**有重量**

### 3.6 战马作为独立单位

- 每匹马有独特属性
- 战斗中可能力竭倒下，骑手需徒步继续战斗
- 若买不起新马，**后续战斗都得步行**——这是真实的长期惩罚

**意义**：把"骑兵"从"加成包"变成"组合单位"。
这是加贺昭三对"载具"在 SRPG 里的最大胆尝试。

### 3.7 精细地形系统

- 每个地形格子针对 **10 种不同职业类别有不同效果**
- 不只是"森林=闪避+15"，而是"森林对骑兵-3 移动 / 对盗贼+10 闪避 / 对法师..."
- 表格庞大但**每一行都有战术意义**

### 3.8 "感觉对了"的 RNG

- 不一定"数学公平"，但体验"恰到好处"
- 法师有现实概率躲过重击，甚至能扛住一次幸运一击
- **反击与反反击充满"可靠的不可预测性"**——让弱单位在对的格子里偶尔英雄一次

**意义**：这是 Berwick 最难复制的体验——它**不靠透明命中率**，而是靠"剧情感"。
玩家事后会说"那一次法师居然躲过了"，而不是"我赌 25% 中了"。

### 3.9 升级系统

- 每次升级数值增长极低（往往只 +1 点）
- 但每次至少**保证有提升**——优于前作 Tear Ring Saga 经常"空升级"

**意义**：把 FE 系传统的"升级 RNG 焦虑"砍到最小。
角色成长慢但**确定**，让玩家把注意力**放在战术决策上**而非"刷级 SL"。

### 3.10 雇佣兵 + 装备稀缺

- 主城 Narvia 散布雇佣兵，**每人都有用**——不是公会里呆板等待
- 武器商店分四家专卖（剑盾 / 矛斧 / 弓箭 / 法术药水）——这是设计摩擦，被批评
- **强力武器全店仅一件**——防止有钱玩家加速突破难度曲线

**意义**：装备稀缺让"该把好武器给谁"成为重要战略抉择，
而不是"全队人手一把神器"。

## 四、为什么好玩（抽象层）

| 体验驱动力 | 具体由什么产生 |
|---|---|
| **战术决定胜负的真实感** | 减法制公式 + 透明数值 + 混合回合 |
| **困难但不沮丧** | 援军节奏控制 + 受伤系统软化死亡 + 任务多样化 |
| **手感对的 RNG** | 反击 / 闪避有戏剧性，但极少"一发翻盘" |
| **战略组合性** | 雇佣兵性格 + 装备稀缺 + 战马独立 |
| **优雅简洁** | 减法公式简单到心算，但通过"特例"产生深度 |
| **多样化任务** | 护送 / 守城 / 反魅惑 / 通缉令——不只"杀光" |
| **道德灰度** | 不是"正义扫除黑暗"，而是双手沾血的真实冲突 |

> 评论家原话：
> "玩家始终相信：只要思考 + 一点运气，总能想出办法。"
> "战术决定胜负"在 Berwick 里不是营销语，而是设计承诺。

## 五、设计缺陷与争议

### 5.1 命中率"反人类"
- 早期低技能 + 低精度武器 = "和敌人差不多的命中率"
- 国外玩家精辟概括为 "luck"——不是策略
- 与"手感对的 RNG"是同一硬币两面：少数人觉得有趣，多数人觉得恼火

### 5.2 减法制的根本困境
- 防御 ≥ 攻击 = 完全打不动
- 这迫使设计师**必须**给重甲反制工具（匕首特例）
- 一旦反制工具失衡，**整个游戏崩盘**——Berwick 通过"匕首低伤+高负伤+力量不计入"做了精密平衡，但很脆弱

### 5.3 商店分四家的设计摩擦
- 给弓箭手买新弩 + 弩箭要跑两家店
- 评论家公认这是不必要的复杂度
- "真实感"不应该以"操作疲劳"为代价

### 5.4 战马倒下 = 永久惩罚
- 设计上很硬核，但**不可恢复的随机损失**违背玩家心理预期
- 部分玩家选择 SL 大法

### 5.5 升级 +1 的体感乏味
- 修复了"空升级"问题，但代价是"升级感"也消失了
- FE 系玩家习惯"升级时心跳加速"，Berwick 把这一情绪点砍掉了

### 5.6 战斗时长偏长（30–60 分钟）
- 单战 30+ 回合常态
- 与 BB 类似，后期"清场"阶段乏味
- 没有快速决斗机制

### 5.7 PS2 独占 + 仅日版
- 没有官方英化，传播极受限
- 西方主流 SRPG 圈对它的认知度极低
- 是"被埋没的杰作"的代名词

### 5.8 加贺昭三的传统包袱
- 永久死亡 + 武器耐久仍存在
- 一些设计选择是**为了"硬核"而硬核**，未必是为了体验最优
- 评论家承认 Berwick "保留了系列硬核传统"，但**这是包袱不是资产**

## 六、对我们项目的启发

### 6.1 直接拿（哲学层）
- ✅ **"算清楚的胜利"哲学**——透明数值，让玩家服气
- ✅ **多样化胜利条件**——不只是"杀光"，可考虑护送 / 守住 N 回合 / 击杀首领
- ✅ **手感对的 RNG**——保留戏剧性，但"高命中率连 miss 两次"必须避免

### 6.2 直接拿（机制层）
- 🟡 **混合回合制**——值得**长期评估**
  - 当前我们用速度排序，已经接近混合回合
  - 但还没做到 Berwick 那种"敌我交替"的透明感
  - **建议**：在战斗 UI 顶部加一条**未来 5 步行动顺序条**（like Divinity / FFT）
- 🟡 **受伤 + 撤退/俘虏系统**——比"死即终结"更柔和，比"无代价复活"更有重量
  - 我们的"匕首处决留俘虏"思路与此一致

### 6.3 警示（避开它的坑）
- ❌ **减法制公式**——我们用三段式（双血条），底层数学结构更适合"硬核但不劝退"
- ❌ **匕首完全无视防御**——我们的双血条下，匕首的"低对甲 + 高穿刺"已经够了
  - 见前几轮讨论：Berwick 的"无视防御"是**减法制的破局钥匙**，
    在我们的三段式里**不是必需品**
- ❌ **商店分四家**——任何"操作摩擦"都不能为"真实感"背书
- ❌ **战马随机倒下不可恢复**——单调的负反馈会逼玩家 SL
- ❌ **升级 +1**——成长曲线要有"心跳点"，不能全平
- ❌ **30+ 回合的战斗**——目标 6–10 回合

### 6.4 真正值得借鉴的"Berwick 精神"
1. **用特例制造深度**——不是堆维度，而是给某些武器/单位**唯一的关键字**
   - 这与我们打算做的"武器关键字制"完全一致
2. **战术决定胜负的承诺**——所有数值都要让玩家**事前可推**
3. **困难但有解** ——援军 / 强敌登场要"早预告 + 给应对工具"，不能空降坑人
4. **任务多样化**比"敌人多样化"更便宜也更有效
