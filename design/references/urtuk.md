## 一、产品定位

| 项 | 内容 |
|---|---|
| **类型** | 开放世界 + 程序生成 + 回合制战术 RPG |
| **开发者** | David Kaleta（独立 / Mad Sheep Studios，单人/极小团队） |
| **EA 启动** | 2020-02-14 |
| **正式发售** | 2021-02-27 |
| **价格** | 约 $14（Steam 美区） |
| **评测** | Steam **特别好评（92%）**，2375+ 评测 |
| **画风** | 全手绘 2D + 黑暗低魔幻 |
| **平台** | Windows / macOS / Linux（PC 单一） |
| **核心受众** | Battle Brothers / Wartales / Darkest Dungeon 同好 |

**一句话**：BB 派的硬核战棋 + 独创"特性提取/吸收"系统 + 可垂直改造的 hex 战场。

## 二、世界观与剧情

- 低魔幻世界：**没有**精灵 / 矮人 / 巨龙；魔法极其轻量
- **巨人族**早已灭绝
- **生命精华（Life Essence）**：从远古巨人骸骨提取的神秘液体，本期望治病/抗衰老，实际多导致**突变或致死**
- 在"疗养院"（Sanatorium）里以平民为实验体
- 主角 Urtuk 从疗养院逃出，自身已严重突变，需在世界寻找解药
- **被砍人评价**：剧情薄弱，玩家普遍认为这是游戏短板，叙事让位给玩法

## 三、核心战斗机制（重点借鉴方向）

### 3.1 基础结构
- **六角格回合制**
- 上场单位 **最多 6 人**（小队精简，强化每个单位重要度）
- 行动顺序：左侧"行动条"按 Speed 滚动展示，最上方单位行动

### 3.2 地形高低差系统 ⭐
**这是 Urtuk 最大的战斗特色**：

| 规则 | 描述 |
|---|---|
| **高度限制** | 近战只能攻击 **±1 高度差** 内的目标 |
| **不可跨越** | 高度 2 → 高度 4 必须经由高度 3 中转 |
| **高地加成** | 高处攻击低处获得命中/伤害加成（敌我通用） |
| **远程封锁** | 高度差 ≥2 时近战完全失效 |
| **地形改造** | **某些职业可主动抬升/降低地形高度**（Javelinier 标枪兵） |
| **死亡坑** | 推敌入坑/尖刺/高崖 → **直接致死**（环境击杀） |

**对 Game001 的启示**：
- 我们 design.md § 5½ 的高低差只做到 +10% 命中，**没有触发"无法攻击"的硬约束**
- Urtuk 的"高差 2 不能近战"是把地形从软修正变成硬规则，**战术权重 ×3**
- 地形主动改造（标枪兵造塔）是独一无二的 build 方向，FFT/BB 都没有

### 3.3 特殊地形效果
- **尖刺**：流血 DoT
- **流沙**：缓速
- **油 + 火**：链式反应（DOS2 派）
- **草丛**：隐蔽

### 3.4 援护机制
- 队友间可设置"援护"
- 敌人攻击我方时友军自动反击
- 远程支援（Ranged Support）：我方近战与敌交战时，远程队友自动追加一发箭
- → **这是 Urtuk 的"军阵"答案**：用援护代替 BB 的 ZoC 借机

### 3.5 组合攻击
- 多种连携攻击（推 + 命中、眩晕 + 远程致命一击等）
- 玩家自由发现 Combo，无固定脚本

## 四、特性 / 突变器（Mutators）系统 ⭐⭐⭐

> **这是 Urtuk 的灵魂机制，也是与所有同类游戏最大的差异化设计**。

### 4.1 核心循环

```
战斗中 → 鼠标悬停敌人 → 看到敌人特性池
↓
战斗胜利 → 选择性"提取（Extract）"敌人特性
↓
得到突变器物品（Mutator）
↓
装备给己方角色 → 获得战斗加成
↓
佩戴 N 场战斗 → 触发"吸收（Absorb）"选项
↓
吸收后突变器消失，特性永久内化
↓
装备槽空出 → 装备新突变器，循环
```

### 4.2 突变器分类

| 维度 | 类型 | 特点 |
|---|---|---|
| **数值缩放** | Static（静态） | 等级仅改变 HP 加成（-10% → +13%）。吸收时不保留等级 |
| **数值缩放** | Scaling（缩放） | 等级直接影响核心数值（治疗、伤害）。**吸收时数值永久保留** |
| **用途** | 普通突变器 | 装备槽：战斗加成 |
| **用途** | Focus Mutator（聚焦突变器） | **替换职业默认 Focus 技能** —— 直接重塑职业玩法 |
| **资源** | Life Stones（生命石） | 合并 → 任意突变器 +2 级 |
| **资源** | Evolution Stones（进化石） | 合并 → 已吸收的缩放型特性 +2 级 |

### 4.3 吸收机制（关键）

| 突变器成本 | 完成吸收所需战斗 | 半程"附着"提示 |
|---|---|---|
| 22 Life Essence | 15 场 | 8 场 |
| 44 Life Essence | 30 场 | 15 场 |
| 88 Life Essence | 45 场 | 23 场 |

**核心规则**：
1. **每个职业有"最大吸收数"上限**（如 Bounty Hunter 4 个 / Berserker 1 个）
2. 取下突变器 → 计数暂停
3. 拒绝吸收 → 该件突变器永久无法被该角色吸收，**但其他不受影响**
4. 缩放型务必升满再吸收（数值永久保留）

### 4.4 经典突变器示例

| 名称 | 类型 | 效果 |
|---|---|---|
| Light Foot（轻足） | 静态 | 移动力 +1 |
| Stalwart（坚毅） | 静态 | 抗推 / 抗位移 |
| Quick Reflexes（快速反应） | **缩放** | 受击后 Speed +X，与狂战极强 Combo |
| Flesh Eater（食肉者） | **缩放** | 击杀后回 X% HP |
| Leeching Criticals（吸血暴击） | **缩放** | 暴击吸血 |
| Blood Lust（嗜血） | 静态 | HP 阈值触发狂暴 |
| Sharp Eyes（鹰眼） | **缩放** | 远程命中 + 暴击 |
| Ranged Support（远程支援） | 静态 | 我方近战交战时自动追射 1 发 |
| Lethal Blow（致命一击） | 静态 | 对眩晕敌人致死 |
| **Devils Pact**（魔鬼契约） | Boss 限定 | 仅 Map 2 Boss 掉落 |
| **Gods Blessing**（神之祝福） | Boss 限定 | 仅 Map 3 Boss 掉落 |

### 4.5 对 Game001 的启示

我们的 `class-system.md` 已经走了"职业 + JP + Wisdom"路线（FFT 派），**Urtuk 的特性提取系统刚好是另一种 build 哲学**：

| 维度 | FFT 派（Game001 当前） | Urtuk 派 |
|---|---|---|
| 进步来源 | 经验 → 升级 → JP → 学技能 | 战斗 → 提取敌人特性 → 装备 → 吸收 |
| Build 自由度 | 职业 + 副职业 + 反应 + 移动 4 槽 | 装备槽 + 已吸收槽（无明确上限） |
| 反 grind | 经验池固定（Berwick 派） | 自然战斗中提取，不需要刻意刷 |
| 决策点 | 升级时选学什么技能 | "这件突变器吸收 vs 留装备 vs 拆解换钱" |

**可借鉴点**：
1. **敌人特性显示**：鼠标 hover 敌人时显示其特性池（增加情报深度）
2. **资源化的特性**：把"被动天赋"变成可流通的物品 → 战利品价值提升
3. **吸收倒计时**：突变器佩戴 N 战才能吸收的设计，给玩家"养成跨时段角色"的体验
4. **反 grind 哲学契合**：Urtuk 的特性获取与战斗深度耦合，而不是经验刷怪 → 与我们 Berwick 派的"EXP 池固定"理念一致

## 五、职业体系

### 5.1 人类职业（首张地图可招募，10 个）

| # | 职业 | 角色定位 | 核心能力 / 装备 |
|---|---|---|---|
| 1 | Spearman（长枪兵） | 顶级坦 / 卡咽喉 | 板甲 + 长枪 / **Spearwall（无限枪墙）** + Taunt |
| 2 | Guardian（守卫） | 队友护盾 / 推坑 | 板甲 + 盾 / **Shield Ally**（完全无效化攻击）+ Bash 推坑/眩晕 |
| 3 | Footman（步兵） | 装甲副坦 / 长武器 | 板甲 + 长武器 / 类 Spearman 但无 Taunt / Ram 冲撞 |
| 4 | Assassin（刺客）⭐ | 高机动侧翼 | 双剑无甲 / 4 移动 + 背刺撤退 / Assassin's Mark / 12 级 Spike Trap |
| 5 | Berserker（狂战士） | 反甲专家 | 双手武器无甲 / Berserker Duelyst（自身打两次但触发反击）/ Shred Armor |
| 6 | BloodKnight（血骑士） | 吸血输出 | Lifesteal Strike（Focus）/ 10 级 Mansplitter 爆发 |
| 7 | Hunter（猎人） | 远程精准射手 | 4 格弓 / Aimed Shot 满 Focus 5 格暴击 / 需视线 |
| 8 | Javelinier（标枪兵）⭐ | 远程 + 地形 | 3 格弧线（无视视线）/ **抬升/降低地形** / Ranged Support 强 Combo |
| 9 | Warmonk（战僧）⭐ | 控制大师 | Jump 跳跃（无视高差）/ Flip 互换位置 / Aegis 反击 / 暴击眩晕 Combo 起手 |
| 10 | Priest（牧师） | 纯辅助 | Lifesteal 给队友 / Aegis（一次免死）/ Feast 抵消消耗 / 极脆 |

### 5.2 敌方/可招募派系（9 大类）

1. **Scavenger（拾荒者）** — 第一张地图主要敌人，与 Bandit 类似
2. **Forsaken（被遗忘者）** — 第 1-2 区中级派系，elites 带 Revenge 特性
3. **Bounty Hunter（赏金猎人）** — 玩家可招募的精英职业
4. **Cultists（邪教）** — 来自疗养院的追捕者，与 Valdor 联手
5. **Swamper（沼泽人）** — 沼泽派系，毒攻 / 鲜血流
6. **Beasts（野兽）** — 唯一不可招募派系；特性池极大且随机，最危险
7. **Werebeasts（兽人）** — 可招募，每隔几天人形/兽形切换
8. **Vampires（吸血鬼）** — 招募后周薪要"血"而非货币；可把队友"转化"
9. **Necroreavers（亡魂收割者）** — 不死系，Bloodless（无血）规避吸血
10. **Valdor** — 高级敌方派系，搭配 Cultists 出现
11. **Others** — 任务/特殊单位

### 5.3 Focus 技能（每职业一个核心技能）
- 每个职业有一个 Focus Ability，蓄力满后释放
- 部分职业还有 **Overfocus**（顶级蓄力技，更强但需更多 Focus）
- **Focus Mutator** 可替换 Focus 技能 → 重塑玩法

### 5.4 招募 / 经济
- **Life Essence** 是核心货币（招募 + 突变器 + 进化石都需要）
- **永久死亡** + **特性继承到突变器**（死掉的队友身上突变器可被回收）
- 装甲单位可在 **Armorer's Guild** 招募，自带 Soldier 特性
- **Vampires 招募成本**：每周一份"血液"工资 + 原本的 Trilium

### 6.1 沙盒结构
- **战争迷雾** + 程序生成
- 三大地图层级（Map 1 / 2 / 3），每张地图都有终局 Boss
- 玩家位置：红色箭头；可移动地点：周围绿色点

- 左下角沙漏 → 快进时间（用于刷事件、等队友养伤）

### 6.3 事件标识
| 标识 | 含义 |
|---|---|
| 蓝色旗帜 | 普通敌人 |
| 白色文字 | 任务（救 NPC 等） |
| 红色骷髅 | 强力敌人 |
| 系统提示 | **风险等级评估**（玩家自评） |

- 村庄初始被敌占领，攻占后 → 玩家收入来源

### 6.5 任务事件
- **Ambushed Alchemist**（被伏击的炼金师）→ Focus Mutator 奖励
- **Scavenger versus Scavenger** → Focus Mutator
- **Driver's Quest**（驯兽师）→ Evolution Stones
- **???（隐藏任务）** → Evolution Stones

## 八、对 Game001 的具体借鉴建议

### 8.1 高优先级（可直接吸收进 Phase 3-4）

1. **地形高度硬约束**：不仅高地 +10% 命中，还要"高差 ≥2 时近战不可达"
   - 写入 `design/terrain-system.md`
   - 让"高差 = 战术墙"而不仅是软修正
2. **死亡坑机制**：尖刺 / 悬崖 / 火坑 → 推/抛入坠物**直接致死**
   - 与 design.md § 5½ 的"推入地形"协同
   - 唐代锚点：城墙、护城河、火盆陷阱、虎口阵
3. **远程支援作为通用天赋**：友军近战交战时自动追射
   - 写进通用辅助技能池"侦察类"或单列"协同类"
   - 替换 BB 缺失的"远程节奏感"

### 8.2 中优先级（Phase 4+ 战略层）

4. **特性提取系统的**唐代化形参**
   - 称作"血脉"或"武学心得"
   - 死敌身上提取**1 件**心得（武学要诀 / 战场习气 / 部族体术）
   - 与现有 Berwick 派"经济紧缺"协同 → 心得物品可买卖
5. **Boss 限定特性**
   - 唐代锚点：将军遗甲 / 西域秘术 / 沙陀血脉 / 道家长生诀
   - 全 campaign 限量，强构筑核心
6. **Focus 技能 + 替换机制**
   - 类似我们已设计的"职业专属能力"，但允许特殊词条**永久替换** Focus
   - 增加 build 多样性

### 8.3 低优先级（远期可选）

7. **6 人小队上限**（vs 我们目前的 8 人）
   - 数据待权衡：6 人节奏更快但容错低
8. **风险等级评估** UI
   - 大地图任务旁边自动给出预估难度（与 Phase 4 接合同系统协同）
9. **Werebeast / Vampire 转化**
   - 唐代锚点：感染 / 道术尸变 / 蛊惑 → 队友异化机制

### 8.4 不建议照搬

- ❌ **完全程序生成的剧情** — 我们 design.md 明确做"半开放主线"，剧情厚度是相对 Urtuk 的差异化优势
- ❌ **手绘 2D** — 我们走"卡通化 + 唐朝丹青"，美术管线已锁
- ❌ **永久死亡 + 仅一条线** — 我们做的是"永死 + 大队伍轮换"，比 Urtuk 容错高一档（避免新手劝退）
- ❌ **战斗模式化导致后期疲劳**（玩家普遍批评）— Urtuk 后期所有战斗都是相似 hex + 突变器构筑，**我们必须用"地形多样性 + Boss 嗜血 + 援军 + 任务目标多样化"对冲**
