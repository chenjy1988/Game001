# 战场交互系统 v1.0

> 本文档承载战场地形与交互设计的完整规则。
>
> 设计血脉：**地形学 TO + 掩护学 XCOM + 环境击杀 Into the Breach + 地形组合 DOS2**。
> 核心目标：地形不是背景板，是战术资源——玩家应主动利用地形获得优势，而非仅靠数值碾压。

---

## 一、设计哲学

1. **地形是第三位玩家**——每张地图的地形分布应显著影响战术决策，而非纯粹的视觉装饰
2. **可交互 > 静态加成**——地形不是"站在上面拿 buff"这么简单，要能被**破坏、改造、利用**
3. **环境击杀是奖励不是主力**——推人下水/火烧草丛是战术甜点，不是通关手段
4. **反割草约束**：所有地形伤害遵守 design.md 红线 #3（DoT ≤ 10/回合，持续 ≤ 3 回合）

### 与其他系统的关联

| 系统 | 关联方式 |
|---|---|
| **职业系统** | 斥候侦察揭示地形/伏击；方士战兵改造地形；位移技能推拉目标进地形 |
| **武器系统** | Crush 攻击可破坏掩体；远程射击被地形遮挡 |
| **气力系统** | 粗糙地形移动气力消耗增加 |
| **命中率公式** | 高地 +10%/低地 -10%；掩体遮挡远程命中 -75% |
| **后勤-道士** | 战场天气预测/改造（大地图联动） |

## 二、地形类型总览（8 大类）

### 2.1 基础地形（影响移动 + 防御）

| 地形 | 移动 AP | 气力额外消耗 | 防御修正 | 视野影响 | 唐朝语境 | 备注 |
|---|---|---|---|---|---|---|
| **平地** | 2 | 无 | 无 | 无 | 平原/官道 | 基线 |
| **草地** | 2 | 无 | +5 eff_def | 无 | 草甸/牧野 | 草丛提供轻微遮蔽 |
| **高地** | 2（+1 爬升） | +1/格 | 高位 +10% Hit / 低位 -10% Hit | +1 视野 | 丘陵/塬上 | 高度差 1-3 级 |
| **森林** | 3 | +2/格 | +10 eff_def | -2 视野（林内）/ 遮挡远程 | 山林/松林 | 远程遮挡 + 高回避 |
| **沼泽/水泽** | 4 | +4/格 | -5 eff_def | 无 | 泽国/河滩 | 极慢+防降低+可困住 |
| **雪地** | 3 | +2/格 | 无 | 无 | 朔方雪原 | 滑溜，疾跑 AP +2 |
| **道路** | 1.5（向下取整=1） | -1/格 | -5 eff_def | 无 | 官道/栈道 | 快速移动但无遮蔽 |
| **浅水** | 3 | +3/格 | -10 eff_def | 无 | 溪流/浅滩 | 移动减半，防御大降 |

### 2.2 掩体/障碍物（可交互，可破坏）

| 障碍物 | 掩体等级 | 生命值 | 可破坏 | 破坏方式 | 阻挡视野 | 阻挡移动 | 唐朝语境 |
|---|---|---|---|---|---|---|---|
| **拒马** | 高掩体 | 60 | ✅ | 任意攻击累计伤害 ≥60 | ✅ 完全 | ✅ 完全 | 拒马桩/鹿角 |
| **木栅/篱笆** | 低掩体 | 30 | ✅ | 任意攻击累计伤害 ≥30 | ✅ 部分 | ✅ 完全 | 寨栅/篱墙 |
| **石墙/巨石** | 高掩体 | ∞（不可破坏）| ❌ | — | ✅ 完全 | ✅ 完全 | 城墙/山岩 |
| **草丛（高草）** | 低掩体 | — | ✅（火烧）| 火系攻击/方术火符 | ✅ 躲人 | ❌ 可穿过 | 芦苇/蒿草 |
| **树木** | 低掩体 | 80 | ✅ | Crush 武器 / 火烧 | ✅ 部分 | ✅ 完全 | 松/柏/槐 |
| **车辆/辎重** | 高掩体 | 50 | ✅ | 任意攻击累计伤害 ≥50 | ✅ 完全 | ✅ 完全 | 辎车/粮车 |
| **营帐** | 低掩体 | 20 | ✅ | 任意攻击 / 火烧 | ❌ 不遮挡 | ✅ 完全 | 军帐/毡帐 |

**掩体等级效果**：
- **高掩体**：远程攻击命中 -75%（XCOM 同款；有 Bullseye/鹰眼天赋 → -50%）
- **低掩体**：远程攻击命中 -35%
- **穿过（不阻挡移动）**：不提供掩体加成，但提供躲藏效果（草丛）

### 2.3 特殊地形（触发效果）

| 地形 | 效果 | 持续 | 可交互 | 交互方式 |
|---|---|---|---|---|
| **火墙/火场** | 站立单位每回合 8 HP 伤害；经过 5 HP | 3 回合 | ✅ 可蔓延 | 方术火符 / 火油罐 / 火烧草丛蔓延 |
| **毒雾** | 站立单位每回合 -5% Hit / -3 eff_def | 2 回合 | ✅ 可点燃 | 方术毒符 / 不良人毒针 / 毒药消耗品 |
| **油面** | 经过滑倒（1 回合 Move=0）；可被点燃成火场 | 永久（直到点燃或清除）| ✅ 可点燃 | 方术火符 / 火箭 / 火油罐 |
| **冰面** | 经过 30% 概率滑倒（本步 AP +2）；水+冰术 | 2 回合 | ✅ 可碎裂 | Crush 攻击碎裂 → 变浅水 |
| **烟雾** | 站立单位远程命中 -50% / 被远程命中 -50% | 2 回合 | ✅ 可驱散 | 风符 / 强风天气 |
| **深水** | 不可通行（悬崖等效）；被推入即坠落伤害 | 永久 | ❌ | — |

## 三、地形交互动作

### 3.1 通用交互动作（任何角色可执行）

| 动作 | AP | 气力 | 效果 | 条件 |
|---|---|---|---|---|
| **翻越障碍** | 2 | 5 | 跳过低掩体到对面格 | 障碍物 ≤ 低掩体等级；weight ≤ 18 |
| **破坏掩体** | 攻击 AP | 正常攻击气力 | 对掩体造成伤害，累计 ≥ 生命值则摧毁 | 掩体可破坏；用攻击动作指向掩体 |
| **使用拒马** | 2 | 3 | 将相邻拒马推到自身与目标之间 | 相邻有拒马/木栅 |
| **推开车辆** | 3 | 8 | 将相邻车辆推 1 格 | weight ≤ 自身 stamina / 15 |
| **点燃地形** | 2 | 3 | 用火油罐/火符点燃目标格 | 装备火源；目标格可燃（草丛/油面/营帐） |
| **伏地隐蔽** | 2 | 2 | 在草丛中进入隐蔽状态（被侦察类技能发现前不可见）| 必须在草丛/森林格 |

### 3.2 职业专属交互

| 职业 | 交互动作 | 效果 |
|---|---|---|
| **方士战兵** | 布火符 / 布水符 / 布风符 | 在目标格创建对应地形效果（火墙/冰面/驱散烟雾），Wisdom 影响范围 |
| **斥候** | 远视 | 临时扩展视野半径 3 格，持续 1 回合；可发现隐蔽单位和伏击 |
| **不良人** | 布陷阱 | 在相邻格放置陷阱（毒针/绳索），踩中触发 debuff |
| **跳荡** | 拆拒马 | 1 AP 拆除相邻拒马（跳荡攻坚先锋定位） |
| **僧兵** | 碎冰 | 长棍攻击冰面 → 冰面碎裂变浅水 |

### 3.3 位移技能与地形交互（详见 class-system.md 位移类技能）

| 位移结果 | 地形效果 |
|---|---|
| 被推入水泽 | 移动 AP 变 4/格 + 每回合需检定才能脱离（Str check vs DC = 地形难度）|
| 被推入火场 | 立即受 5 HP 伤害 + 站立受持续 8/回合 |
| 被推入拒马 | 15 HP 伤害 + 停在拒马前 |
| 被推入深水/悬崖 | 坠落伤害 = 20-40 HP（视高度差） |
| 被推撞墙壁 | 5 HP 伤害 + 停住 |
| 被推撞另一单位 | 双方各 3 HP 伤害 + 均停住 |

## 四、地形组合与链式反应（DOS2 灵感，简化版）

**设计原则**：DOS2 的元素组合极度复杂（水+电=麻痹蒸汽云、油+火=爆炸），本作做**简化 5 条链式反应**，不追求物理模拟，追求战术可读性。

| 组合 | 触发条件 | 结果 | 战术价值 |
|---|---|---|---|
| **油 + 火** | 油面格被火攻击命中 | 油面变为火场（3 回合），范围扩大 1 格 | 封锁通道 / 逼迫走位 |
| **草丛 + 火** | 草丛格被火攻击命中 | 草丛燃烧 → 火场（3 回合），蔓延到相邻草丛（1 回合延迟）| 驱赶隐蔽单位 / 区域封锁 |
| **水 + 冰** | 浅水格被冰术命中 | 浅水变为冰面（2 回合）| 制造滑倒区 / 阻断追击 |
| **冰 + 碎** | 冰面被 Crush 攻击命中 | 冰面碎裂 → 变为浅水 | 僧兵专长 / 反制冰面 |
| **毒雾 + 火** | 毒雾格被火攻击命中 | 毒雾引爆 → 范围 10 HP 伤害 + 毒雾消散 | 一次性爆点 vs 持续压制，玩家选择 |

**反割草约束**：
- 链式反应伤害 ≤ 10 HP（遵守红线 #3）
- 蔓延延迟 1 回合 → 给玩家反应时间
- 链式反应不叠加（油+火 ≠ 油+火+火 = 两次伤害，只算一次引爆）

## 五、草丛隐蔽系统

**核心设计**：草丛是斥候/游侠/不良人的核心战术工具，不是全队的通用躲藏点。

### 5.1 隐蔽条件

1. 当前格为草丛/森林
2. 执行"伏地隐蔽"动作（2 AP / 2 气力）
3. 进入隐蔽状态

### 5.2 隐蔽效果

- **对远程单位**：完全不可见（无法被远程攻击指定）
- **对近战单位**：相邻时可见（距离 1 自动发现）；非相邻需侦察检定
- **侦察检定**：`观察者 Wisdom × 0.5 + 30` vs `隐蔽者 Init × 0.3 + 10`（每回合自动 roll）

### 5.3 破除隐蔽

| 方式 | 条件 |
|---|---|
| 被相邻敌方单位发现 | 距离 1 自动发现 |
| 侦察检定成功 | 上述公式 |
| 隐蔽者主动攻击 | 攻击后自动暴露 |
| 火烧草丛 | 草丛变火场 → 隐蔽者暴露 + 受伤 |
| 位移技能拉出 | 擒拿拉拽隐蔽者 → 暴露 |
| 声音事件 | 隐蔽者移动（非潜行）→ 周围 3 格敌方侦察检定 +20 |

### 5.4 职业差异

| 职业 | 隐蔽优势 |
|---|---|
| 斥候 | "潜伏"被动：隐蔽时侦察检定对抗 +15；伏地隐蔽 AP-1 |
| 游侠 | "暗影步"：从隐蔽状态攻击 +10% Hit |
| 不良人 | "无声杀"：从隐蔽状态攻击首次暴击率 +20% |
| 其他 | 正常隐蔽，无额外加成 |

## 六、拒马系统

**设计背景**：唐代军阵常用拒马桩/鹿角作为阵前防御工事。拒马是本作最独特的地形元素——可部署、可破坏、可被跳荡拆除。

### 6.1 拒马属性

| 属性 | 值 |
|---|---|
| 掩体等级 | 高掩体（远程 -75%） |
| 生命值 | 60 |
| 阻挡移动 | ✅ 不可通过 |
| 阻挡视野 | ✅ 完全阻挡 |
| 可部署 | ✅ 战前布阵阶段放置 |

### 6.2 拒马交互

| 交互 | 效果 |
|---|---|
| **被推入拒马** | 15 HP 伤害 + 停在拒马前格 |
| **冲锋撞拒马** | 冲锋者 + 目标各受拒马剩余 HP × 0.3 伤害；拒马受冲锋伤害 |
| **跳荡拆除** | 1 AP 拆除（跳荡职业专属，攻坚先锋定位） |
| **Crush 攻击** | 对拒马伤害 ×1.5（战锤/战斧砸桩） |
| **火烧拒马** | 拒马 3 回合内 HP 每回合 -20 → 3 回合后自动摧毁 |

## 七、高地系统

### 7.1 高度规则

- 高度分 0-3 级（平地/低丘/高丘/绝壁）
- 相邻格高度差 ≤ 1 可直接移动
- 高度差 ≥ 2 不可直接移动（需寻路）
- 每爬升 1 级 +1 AP 移动消耗（Pathfinder/寻路者天赋抵消）

### 7.2 高地战术修正

| 状态 | 近战命中 | 远程命中 | 远程射程 | 效防 | 备注 |
|---|---|---|---|---|---|
| 高 1 级 vs 低 | +10% | +10% | +1 | +5 | 制高点优势 |
| 高 2 级 vs 低 | +10% | +20% | +2 | +5 | 绝壁射台 |
| 低 1 级 vs 高 | -10% | -10% | 不变 | -5 | 仰攻劣势 |
| 高度差 ≥ 2 | 无法近战 | 正常射击 | +2 | — | 城墙/悬崖 |

### 7.3 坠落

被位移技能推下高地 → 坠落伤害：
```
坠落伤害 = 高度差 × 15 + rand(0, 10)
高度差 1 → 15-25 HP
高度差 2 → 30-40 HP
高度差 3 → 45-55 HP
```
- 坠落伤害直接扣 HP（无视护甲）
- 坠落后落在低处格，若落点被占则偏移到最近空格

## 八、水泽/沼泽系统

### 8.1 水泽类型

| 类型 | 移动 AP | 防御修正 | 效果 |
|---|---|---|---|
| **浅水** | 3 | -10 eff_def | 减速；可被冰术冻成冰面 |
| **沼泽** | 4 | -5 eff_def | 极慢；每回合开始需 Stamina 检定（DC = 地形难度）失败 → 困住 1 回合 |
| **深水** | 不可通行 | — | 等效悬崖；被推入 = 坠落伤害 |

### 8.2 困住机制

```
每回合开始时，在沼泽中的单位做脱困检定：
脱困成功率 = 50% + (stamina_ratio × 30%) + (weight ≤ 12 ? +10% : 0)

失败：本回合 Move = 0（仍在沼泽中，可攻击/使用技能）
成功：本回合正常移动
```

**战术意义**：
- 重甲单位在沼泽中脱困率更低（stamina 消耗快 → ratio 低 → 脱困率低）
- 轻甲单位仍有一定概率被困住
- 沼泽 + 远程压制 = 经典克制重甲冲锋的组合

## 九、战场生成规则

### 9.1 地形权重

| 战场类型 | 地形倾向 | 典型组合 |
|---|---|---|
| **平原野战** | 平地 60% + 草丛 20% + 道路 10% + 浅水 10% | 开阔 + 少掩体 + 骑兵优势 |
| **山地隘口** | 高地 40% + 石墙 20% + 道路 10% + 森林 30% | 逐层争夺 + 拒马封口 |
| **泽国伏击** | 沼泽 40% + 浅水 20% + 草丛 30% + 树木 10% | 斥候关键 + 轻甲优势 |
| **城寨攻防** | 石墙 30% + 木栅 20% + 拒马 15% + 营帐 15% + 平地 20% | 攻坚 + 跳荡拆除 |
| **森林遭遇** | 森林 50% + 草丛 20% + 树木 15% + 浅水 15% | 视野受限 + 隐蔽/伏击 |
| **雪原行军** | 雪地 60% + 冰面 20% + 平地 20% | 滑溜 + 疾跑风险 |

### 9.2 可交互元素密度

- 每张地图至少 3-5 个可交互掩体/障碍物
- 至少 1 个可被火烧/引爆的地形元素（草丛/油面/营帐）
- 拒马仅在城寨攻防/山地隘口出现（2-4 个，战前可部署）
- 高地至少有 2 个可攀爬路径（防止被远程完全压制）
