# 状态效果系统（Buff / Debuff / 临时伤势 / 永久伤势）

> 设计哲学：少而精。每个状态效果必须服务于核心机制（气力、先攻、命中减法制、士气），不凭空造系统。
> 临时 debuff 遵守红线：DoT 每回合 ≤10 伤害，持续 ≤3 回合。
> 伤势分两层：临时伤势（战斗内触发，脱战后保留直到治愈）+ 永久伤势（HP 归 0 触发）。

## 一、Buff（8 种）

| 名称 | 效果 | 持续 | 来源 |
|---|---|---|---|
| **盾墙** | 格挡率 +20% | 1 回合 | 盾牌系主动防御技能 |
| **蓄力** | 下次攻击伤害 ×1.3，消耗后消失 | 至下次攻击 | 陌刀/长矛蓄力出招 |
| **破甲** | 下次攻击护甲穿透 +30%（无视 30% armor），消耗后消失 | 至下次攻击（最多 2 次）| 锤兵/钝器职业技能 |
| **愤怒** | 伤害 +5%，命中 +5% | 2 回合 | HP < 50% 被动触发，或技能赋予 |
| **激励** | 周围友军命中 +10% | 1 回合 | 指挥类技能（押衙亲兵/虞候） |
| **振奋** | 命中 +10% / 防御 +5% | 战斗内 | 士气档位「振奋」（见 §三） |
| **急速** | Init +15 | 2 回合 | 方术类技能 |
| **包扎** | 减轻临时伤势效果（见伤势系统） | 至战斗结束 | 急救技能 / 医户 |

### 包扎 buff 详解

包扎是唯一能减轻临时伤势的 buff，由急救技能或医户职业施加：

| 包扎等级 | 效果 | 来源 |
|---|---|---|
| **简易包扎** | 临时伤势效果 -50% | 任何单位自带急救技能（1 AP） |
| **专业包扎** | 临时伤势效果 -80% | 医户 / 僧兵专业技能 |
| **战场手术** | 移除 1 个临时伤势 | 医户高级技能（3 AP + 消耗药材） |

> 注意：包扎只减轻/移除**临时伤势**，对永久伤势无效。

## 三、士气系统（5 档）

> **五档设计**：在「动摇」与「崩溃」之间增加 **惊惧**，拉长恐惧流推进距离，避免 4 档下 1～2 次压制即崩盘。
> 士气修正经 `CombatModifier` 汇总：**命中为加减百分点；防御为百分比乘算**（非 flat 绝对值）。

### 3.1 档位与战斗修正

| 档位 | id | 命中 | 防御 | 其他 | 典型来源 |
|---|---|---|---|---|---|
| **振奋** Inspired | `morale_inspired` | +10% | +5% | — | 杀敌 / 友军杀敌 / 指挥技能 |
| **稳定** Steady | `morale_steady` | 0 | 0 | 基线 | 战斗开始默认（好感正常时） |
| **动摇** Shaken | `morale_shaken` | -10% | -5% | 不能反击；Move -1 | 友军阵亡目击 / HP&lt;25% / 被瞄头命中 |
| **惊惧** Afraid | `morale_afraid` | -15% | **-10%** | 不能反击；Move -1 | 士气检定失败连降；战吼/枪锋慑敌等恐惧流推进 |
| **崩溃** Broken | `morale_broken` | **-25%** | **-25%** | **不能反击；不能借机攻击** | 处决威慑；惊惧状态下再失败；Resolve 连续检定失败 |

**防御百分比实现**：`eff_defense × (1 + defense_pct)`，例：防御 -10% → `defense_pct = -0.10`。
**不能反击 / 不能借机攻击**：崩溃档同时关闭主动反击与 ZoC 借机攻击；动摇、惊惧仅关闭反击。

### 3.2 开战初始档位（好感）

雇佣兵对玩家的隐藏 **好感度**（`camp-events-system.md` §五）在 `BattleScene` 开战时写入初始 `morale_tier`：

| 好感 | 开局档位 |
|---|---|
| 75～100 | 稳定（80+ 可 20% 振奋） |
| 40～74 | 稳定 |
| 20～39 | **动摇** |
| 0～19 | **惊惧** |

与战斗内升降 **叠加**；仅友军雇佣兵适用，敌军 / 剧情友军按关卡表。

### 3.3 升降规则

| 事件 | 士气变化 |
|---|---|
| 好色 `lustful` **命中**异性且 HP 伤害 &gt; 0 | **50% 概率** 攻击者 **-1 档**（每回合限 1 次）；敌我对称；**日志必写**（见 `personality-system.md` §2.2.1） |
| 击杀敌人 | +1 档（上限振奋） |
| 友军击杀（目击范围内） | +1 档（上限振奋） |
| 友军阵亡（周围 2 格） | Resolve 检定，失败 **-1 档** |
| HP &lt; 25% | Resolve 检定，失败 **-1 档** |
| 被瞄头/爆头命中 | Resolve 检定，失败 **-1 档** |
| 战吼压阵 / 枪锋慑敌等 | 按技能概率 **-1 档**（同目标每回合限 1 次） |
| 处决威慑（虞候/军法） | 检定失败可 **直送崩溃**（跨档） |
| 振奋军心 / 督战斩 / 乐师曲目 | +1 档或阻止本回合下降（按技能） |

> 常规下降每次 **1 档**：稳定→动摇→惊惧→崩溃。恐惧流主战场在 **动摇～惊惧**；打到 **崩溃** 仍需处决威慑或连续压制。

### 3.4 Resolve 检定（框架）

```
检定成功率 = clamp(50% + (defender_resolve - threat_severity) × 1%, 10%, 90%)
```

- `threat_severity`：普通事件 0；目睹友军阵亡 15；Boss 嗜血层数等可叠加。
- 连续 2 次检定失败：额外 **-1 档**（加速滑向惊惧/崩溃）。
- 实现：`Stats.morale_tier`（0=稳定 … 4=崩溃）+ `MoraleSystem`（Phase 2.5 H2）。

## 四、临时伤势（10 种）

> 触发条件：**被攻击时 HP 低于一定阈值**，按伤害类型和命中部位触发。
> 临时伤势**脱离战斗后仍然保留**，需要通过包扎或营地休息治愈。
> 同一单位可同时承受多个临时伤势。

| 伤势 | 效果 | 触发 |
|---|---|---|
| **脑震荡** | 命中 -25%，视野 -1 | 钝击 → 头部 |
| **断臂** | 伤害 -50% | 钝击/斩击 → 身体 |
| **断腿** | 每格移动 +2 AP，Init -40% | 钝击 → 身体 |
| **割伤手臂** | 伤害 -15% | 斩击 → 身体 |
| **割伤腿肌** | 每格移动 +1 AP，Init -20% | 斩击 → 身体 |
| **割喉** | 流血 3 回合 | 斩击 → 头部 |
| **刺穿肺部** | 气力消耗 +50% | 穿刺 → 身体 |
| **刺穿眼眶** | 命中 -50%，视野 -1 | 穿刺 → 头部 |
| **脱臼** | AP 上限 -2 | 钝击 → 身体 |
| **撕裂腹部** | 每回合 -5% 当前 HP | 斩击/穿刺 → 身体 |

### 临时伤势触发条件

**伤害门槛**：单次伤害 ≥ 最大 HP 的 **10%**（任何 HP 状态都能触发判定）

**触发概率公式**：
```
基础概率 = 单次伤害 / 最大 HP × 100%

HP 状态修正：
  HP > 80%（满血）：基础概率 × 0.5    ← 健康状态扛得住
  HP 50%-80%（中血）：基础概率 × 1.0  ← 基线
  HP < 50%（重伤）：基础概率 × 1.5    ← 累积伤害易触发

最终概率 = min(基础概率 × HP修正, 75%)  ← cap 75%
```

**触发例子**（角色最大 HP 100）：

| 当前 HP | 单击伤害 | 基础概率 | HP 修正 | 最终概率 |
|---|---|---|---|---|
| 95（满血）| 15 | 15% | ×0.5 | **7.5%** |
| 95（满血）| 30 | 30% | ×0.5 | **15%** |
| 70（中血）| 20 | 20% | ×1.0 | **20%** |
| 40（重伤）| 20 | 20% | ×1.5 | **30%** |
| 30（重伤）| 30 | 30% | ×1.5 | **45%** |
| 20（垂死）| 50 | 50% | ×1.5 | **75%（封顶）** |

**设计意图**：
- 满血目标被一击重创也可能受伤（物理直觉合理，不再是"血条 50% 硬门槛"）
- 残血状态下伤势触发率显著上升（累积创伤效应）
- 重武器单击 ≥10% HP 伤害是常态，刺客小击数难以单独触发伤势
- 与"残忍"被动协同：刺客 build 依赖前排队友打出伤势再补刀（团队战术）

### 临时伤势恢复

| 方式 | 效果 |
|---|---|
| **简易包扎**（战斗内） | 伤势效果 -50% |
| **专业包扎**（战斗内） | 伤势效果 -80% |
| **战场手术**（战斗内） | 移除 1 个临时伤势 |
| **营地休息**（脱战） | 1 场休息 = 随机移除 1 个临时伤势 |
| **城镇治疗**（脱战） | 移除所有临时伤势，花费 50-200 金 |

## 五、永久伤势（12 种）

> 触发条件：**HP 归 0 时，按击杀来源从对应触发池随机抽取 1 项**。
> 一次 HP 归 0 只触发 1 个伤势。战斗后不自动恢复，需回营地休息 1-3 场战斗。

### 击杀来源分类

| 击杀来源 | 类别 | 说明 |
|---|---|---|
| 被钝器攻击击杀 | 直接攻击-钝击 | 按实际命中部位 |
| 被利器攻击击杀 | 直接攻击-斩击 | 按实际命中部位 |
| 被矛/标攻击击杀 | 直接攻击-穿刺 | 按实际命中部位 |
| 被火药/燃烧瓶击杀 | 直接攻击-燃烧 | 按实际命中部位 |
| 流血 DoT 致死 | DoT-流血 | 失血性休克，逻辑自洽 |
| 毒性 DoT 致死 | DoT-中毒 | 毒素侵蚀内脏 |
| 燃烧 DoT 致死 | DoT-燃烧 | 烧伤并发症 |
| 撕裂腹部 DoT 致死 | DoT-内出血 | 内出血休克 |
| 地形陷阱/无归属 | 其他 | 意外伤害 |

> **实现方式**：每个 DoT debuff 挂载 `damage_type` 标签，tick 时写入单位的 `last_damage_source`；HP 归 0 时读取该字段，从对应触发池抽取永久伤势。

### 永久伤势一览

| 伤势 | 效果 | 触发条件 |
|---|---|---|
| **脑损伤** | Resolve +15% / Wisdom -15% / 命中 -15% | 钝击/燃烧 → 头部 |
| **肘关节断裂** | 武器伤害 -20% / 近防 -30% | 钝击 → 身体 |
| **膝盖断裂** | 近防/远防 -40% / Init -40% | 钝击/斩击/穿刺 → 身体 |
| **肺塌陷** | 气力上限 -20 | 穿刺/燃烧 → 身体 |
| **残废脚** | 每格移动 AP +1 / Init -20% | 任意 → 身体 |
| **缺失耳** | Init -10% | 斩击 → 头部 |
| **缺失眼** | 命中 -25% / 视野 -1 | 斩击/穿刺/钝击 → 头部 |
| **缺失手指** | 武器伤害 -5% | 斩击/穿刺 → 身体 |
| **缺失手** | 无法装备双手武器/盾牌 | 斩击 → 身体 |
| **缺失鼻** | 气力上限 -10 | 任意 → 头部 |
| **精神创伤** | Resolve -20 / Init -30% | 穿刺/钝击 → 头部 |
| **心脏衰弱** | HP 上限 -30% | 钝击/穿刺 → 身体 |

### 触发池（直接攻击）

按伤害类型 × 命中部位查表：

| 伤害类型 | 头部触发池 | 身体触发池 |
|---|---|---|
| 钝击（锤/杖）| 脑损伤 / 缺失眼 / 精神创伤 | 肘关节断裂 / 膝盖断裂 / 心脏衰弱 / 精神创伤 |
| 斩击（刀/剑）| 缺失眼 / 缺失耳 | 缺失手指 / 缺失手 / 膝盖断裂 |
| 穿刺（矛/标）| 缺失眼 / 精神创伤 | 肺塌陷 / 缺失手指 / 膝盖断裂 / 心脏衰弱 |
| 燃烧（火药）| 脑损伤 | 肺塌陷 / 膝盖断裂 |

### 触发池（DoT 致死）

DoT 致死不区分部位，只有身体触发池：

| DoT 类型 | 触发池 | 设计理由 |
|---|---|---|
| **流血** | 心脏衰弱 / 肺塌陷 | 失血性休克 → 器官衰竭 |
| **中毒** | 肺塌陷 / 心脏衰弱 / 精神创伤 | 毒素侵蚀内脏 |
| **燃烧** | 缺失眼 / 缺失鼻 / 缺失耳 | 火烧面部 → 五官烧毁 |
| **内出血** | 心脏衰弱 / 肺塌陷 | 内出血 → 器官衰竭 |
| **其他**（地形等）| 膝盖断裂 / 残废脚 / 心脏衰弱 | 意外伤害 → 肢体/内脏损伤 |

> **设计备注**：
> - 流血/内出血致死只触发内脏衰竭（心脏/肺），逻辑最紧——失血不会砍断肢体
> - 燃烧致死是唯一能触发面部"缺失"类伤势的 DoT——火烧面部烧毁五官，合理
> - 中毒致死可触发精神创伤（毒素损伤神经系统）

### 特殊备注

- **脑损伤**是唯一有正面效果的永久伤势（Resolve +15%，经历生死后更刚），但代价是智慧和精准度下降
- **缺失手**是最严重的伤势——直接限制装备选择，强制改变 build
- **肺塌陷 / 缺失鼻**与气力上限挂钩，重甲 build 受损最大（气力本来就紧张）
- 恢复方式：回营地休息，缺席 1-3 场战斗后伤势恢复（缺席场次由伤势严重程度决定）

> **临时伤势 vs 永久伤势的区别**：
> - 临时伤势：HP < 50% 时被重击触发，脱战后保留，可通过包扎/休息/城镇治疗恢复
> - 永久伤势：HP 归 0 时触发，需长期休息（缺席 1-3 场战斗）恢复
