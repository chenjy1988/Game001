# 经济系统设计 v1

> **设计血脉**：Berwick Saga（资源紧缺核心）+ Battle Brothers（雇佣兵团运营层）+ XCOM（任务奖励驱动）
>
> **核心哲学**：玩家永远在算账。每一文钱、每一点 JP、每一把武器都是稀缺资源。
> "够用"是赢，"宽裕"是奖励，"破产"是常态。

---

## 一、设计哲学：稀缺性是引擎

### 1.1 为什么稀缺性 = 可玩性

```
绝大多数 SRPG：刷 = 强（你时间多就能 grind）
本项目：资源固定，必须做战略取舍
```

**对比表**：

| 维度 | 传统 SRPG（FFT/FE）| 本项目（Berwick 路线）|
|---|---|---|
| EXP 来源 | 可通过刷怪/支线无限获得 | **每场战斗 EXP 池固定**，无法 grind |
| 金钱来源 | 终游戏前会变富裕 | **永远紧缺**，全程算账 |
| 装备来源 | 通关后大量获得 | **战利品 + 商店** 都受限 |
| 雇佣兵来源 | 关卡解锁/支线 | **付费招募 + 死了归零** |
| 玩家心态 | "练满最优 build" | **"我的雇佣兵团能撑过这周吗"** |

→ **稀缺性 = 决策密度 = 可玩性**。

### 1.2 三层稀缺机制

```
层 1：每场战斗的稀缺（短期）
  → 这场打谁？喂 EXP/JP 给谁？带哪几把武器？
  
层 2：每周/每月的稀缺（中期）
  → 这个月接什么合同？修哪几把武器？招新人还是省钱？
  
层 3：整周目的稀缺（长期）
  → 我团队的 6 个人能 Mastered 12 个职业吗？还是只能 6 个？
```

→ **每个时间尺度都有"算账"决策**——这是 Berwick 的精髓。

## 二、EXP / JP 系统（学 Berwick）

### 2.1 EXP 与 JP 是同一种东西吗？

**不是**。我们用两套：

| 资源 | 作用 | 来源 |
|---|---|---|
| **EXP（经验）** | 提升雇佣兵基础属性（HP/Init/Melee 等）| 击杀/参与战斗 |
| **JP（职业点）** | 解锁职业技能/Trait | 击杀/参与战斗（按职业积累）|

→ 一场战斗同时给 EXP 和 JP，但分别记账。

### 2.2 EXP 池固定（核心稀缺）

**主线关卡**：
```
每个主线关卡的总 EXP = 关卡敌人数 × 每个敌人 EXP（固定）

主线 12-15 章 × 平均 8 关/章 × 每关 ~120 EXP 池 = 全游戏总 EXP ~12000-18000

→ 6 个雇佣兵平均每人能拿 ~2000-3000 EXP
→ 不够每个人都满级（顶级单位需要 ~4000 EXP）
→ 玩家必须选择"主力 vs 辅助"——主力满级，辅助够用
```

**支线合同**（Phase 4 落地）：
```
每个合同的 EXP 给少量（30-80）
但合同有限（每周 2-3 个浮现，过期消失）
→ 不能"刷支线"，但比 Berwick 略多自由度
→ 折中方案：既保证稀缺，又给玩家"我有掌控感"
```

### 2.3 JP 池固定（角色一生上限）

**单角色一生 JP 上限 ~3000**：
```
每场战斗 JP：
  普通战斗：30-60 JP
  困难战斗：80-120 JP
  BOSS 战：150-250 JP
  支线合同：20-50 JP

一生总战斗数 ~50-80 场（不死的话）
→ 总 JP ~3000

单职业 Mastered 成本 ~1500 JP
→ 一辈子能 Mastered 2 个职业 + 副职业 1 个 Trait
```

**死亡归零**：
- 雇佣兵阵亡 → 该角色所有 EXP/JP 永久归零
- 新招募 → 从 Lv1 重新积累
- **保护雇佣兵** = 保护投入

→ 这是雇佣兵团运营层的核心张力。

### 2.4 EXP 分配规则（防止"练废"）

```
战斗内 EXP 分配（学 FE/Berwick）：
  
  击杀者获得：60% EXP（鼓励"杀刀人"战术）
  辅助者获得：30% EXP（命中但未击杀）
  全队分配：10% EXP（保底，让坐板凳的也长经验）

防止虐废：
  - 等级差 5+ 时，高等级获得 EXP -50%
  - 同等级击杀强敌：+30% EXP 加成
  - "新兵"杀强敌：+50% 加成（鼓励练新人）
```

→ FE 同款"挑刀人"战术：让需要升级的人击杀关键敌人。

## 三、金钱系统（学 Berwick + 战兄弟）

### 3.1 收入结构

| 来源 | 单次金额 | 频率 |
|---|---|---|
| **关卡基础奖励** | 80-200 | 每关 |
| **关卡 S 级奖励** ⭐ | 300-800 | 每关，需触发条件 |
| **击杀掉落** | 5-30 | 频繁但少 |
| **战利品** | 50-300 | 看运气 |
| **支线合同** | 200-500 | 每周 2-3 个 |
| **任务委托大头** ⭐ | 800-2000 | 章节级，限定 |
| **赌博/抢劫**（高风险）| ±500 | 玩家可选 |

→ **基础收入只够温饱**，**S 级和任务委托才是富裕来源**。

### 3.2 S 级条件示例（Berwick 同款）

每个关卡有"额外目标"，达成给 2-3x 奖励：

```
主线关卡示例："守住西门 5 回合"
  基础奖励（300 金）：守住 5 回合，活到通关
  
  S 级奖励 +500 金：
    ✅ 全员存活（无人受重伤）
    ✅ 没有友军 NPC 死亡
    ✅ 不让首领逃跑
    ✅ 占领特定地点
    ✅ 用某种特定打法（如：无远程伤害）
    ✅ X 回合内完成（提前结束 +30 金/回合）
    ✅ 俘虏 ≥N 个敌人（活捉换额外赏金）
    ✅ 不杀某个敌人（隐藏剧情条件，留活口审讯）
```

→ **每个关卡都有"挑战式打法"**——玩家可选"快速通关"或"精雕细琢"。
→ 这是**重玩价值**的核心机制。

### 3.3 支出结构

| 类型 | 单次金额 | 频率 |
|---|---|---|
| **雇佣兵周薪** ⭐ | 总计 200-800/周 | 每周固定 |
| **武器购买** | 100-3000 | 偶发大笔 |
| **武器修理** ⭐ | 武器价 30-50% | 每 5-15 场战斗 |
| **道具补给**（药水/解毒）| 5-30/件 | 每场战斗后 |
| **招募新人** | 200-2000 | 一次性 |
| **死亡赔偿**（家属抚恤）⭐ | 100-500 | 死亡后强制 |
| **训练费**（提升属性）| 50-200 | 偶发 |
| **营地维持**（食物/马匹/医药）| 100-300/周 | 每周 |

→ **每周固定支出 ~500-1500**——必须接合同养团。

### 3.4 工资递增机制（学战兄弟）

雇佣兵升级 → 工资上涨：
```
Lv 1: 50 金/周（新兵）
Lv 5: 100 金/周（老兵）
Lv 10: 200 金/周（精英）
Lv 15: 400 金/周（明星）
Mastered 上级职业: +200 金/周
```

→ **越练越贵**——精英雇佣兵开销巨大，**死了等于赔大笔投入**。
→ 玩家心理："这个明星我不能让他死"——这就是 RPG 的情感锚点。

## 四、俘虏经济（学 Berwick + 唐朝军事真实）

> **核心**：俘虏 ≠ 击杀。活捉敌人能换取金钱 + 完整装备 + 特定物品，是雇佣兵团的额外收入支柱。
> 不做"招降为团队成员"（避免说客职业的复杂度）—— 俘虏只用来"换"。

### 4.1 设计哲学

```
战兄弟没有俘虏系统，但唐朝军事文化里：
  - 俘虏首领押送官府 → 大笔赏金（"献俘"）
  - 俘虏外族贵族 → 政治筹码 + 赎金
  - 俘虏精英士兵 → 卖给军营做苦工

→ 这是历史真实的雇佣兵收入来源
→ "杀光"不是最优解 —— "活捉"才是真高手
```

### 4.2 战斗结局对装备 / 金钱的影响

| 战斗结局 | 装备掉落率 | 装备耐久 | 金钱奖励 |
|---|---|---|---|
| **击杀（致命一击）** | 30-50%（部分装备损坏）| 与敌人剩余 HP 比例挂钩（残血 = 装备破破烂烂）| 标准 |
| **击晕（非致命）** | 60% | 中等 | 标准 |
| **俘虏 ⭐** | **100%（完整套装）** | 满耐久 | **2-3x（押送给指挥部 / 卖买家）** |
| **逃跑放走** | 0% | — | -50%（声望可能上升）|

→ **俘虏 vs 击杀的经济差距 4-5 倍**——给玩家强烈动机活捉。

### 4.3 俘虏的触发机制

**战斗中产生俘虏的条件**：

```
任一满足：
  ① 敌人 HP 降到 1-15% 时玩家选择"留活口"（非致命攻击模式）
  ② 用"控制类技能"使敌人 0 行动（道士定身符 / 军师奇袭令等）
  ③ 用钝击武器（crush）+ 瞄头组合 → 命中后 30% 概率自动昏迷
  ④ 关卡结束时所有"残血躺地"未死敌人自动转俘虏（仅完全胜利时）
```

→ 玩家需要"控制击杀强度"——这是战斗中的微决策。

### 4.4 俘虏使用方式（简化版，不做招降）

```
战斗结束后玩家选择：

① 押送给指挥部（默认）：
   - 普通士兵：100-300 金 / 人
   - 军官：500-1500 金
   - 高级军官 / 名将：2000-5000 金 + 特殊装备
   - 装备 100% 完整保留

② 卖给买家（特定场景）：
   - 边境奴隶市场：0.7-0.8x 押送价（即时金钱，不需要回大本营）
   - 敌对势力：1.5x 押送价（出卖但有政治后果）

③ 释放（拷问后）：
   - 揭示下一关地图 / 敌人配置
   - 或揭示隐藏支线
   - 0 金钱但获取情报

❌ 不做：招降为团队成员 —— 复杂度过高，专注核心玩法
```

### 4.5 俘虏受职业 / 性格影响

> **仅友军雇佣兵**有性格；敌方为战法倾向，不参与俘虏经济。性格定义见 **`personality-system.md`** §三（1 人 1 性格）。

```
"贪财"（greedy）：
  → 俘虏行动消耗 AP -1（更主动俘虏）

"残忍"（cruel）：
  → AI 偏好击杀残血/溃逃（俘虏率降低）
  → 玩家："我这名雇佣兵不能用来抓人"

→ 性格与经济系统深度联动
```

### 4.6 与 S 级奖励联动

```
关卡 S 级条件示例（更新版）：
  ✅ 俘虏 ≥3 个敌人 → +800 金
  ✅ 俘虏关卡首领 → +2000 金 + 特殊装备
  ✅ 俘虏特定 NPC（剧情触发）→ 解锁支线

→ 俘虏成为 S 级奖励的核心来源之一
```

## 五、战利品掉落 + 主动拾取（学 Berwick + DOS2）

> **核心**：战利品掉在地面，玩家可主动消耗 AP 拾取，或战斗结束自动结算。
> 完全胜利 = 全部获得 / 撤退 = 部分概率 / 失败 = 全部失去。

### 5.1 战利品类型

| 类型 | 掉落位置 | 拾取方式 |
|---|---|---|
| **武器** | 敌人死亡 / 俘虏格 | 走过去 1 AP 拾取 |
| **护甲** | 敌人死亡 / 俘虏格 | 同上 |
| **金钱袋** | 敌人死亡格 | 同上（可叠加多个）|
| **道具**（药水 / 解毒）| 敌人死亡格 / 战利品箱 | 同上 |
| **任务道具**（信件 / 钥匙）| 关卡触发 | 同上 |
| **特殊装备**（神兵）| BOSS 战 / 隐藏战利品箱 | 同上（罕见）|

### 5.2 主动拾取规则

```
战斗中玩家走到地面物品格：
  ① 自动框选物品（UI 提示）
  ② 玩家按"拾取"按钮 → 消耗 1 AP → 100% 获得 ⭐
  ③ 多个物品同格 → 一次 AP 拾取所有
  ④ 选择"忽略" → 物品保留，可下次再来
  
→ 玩家决策：
  "我先打完这个敌人 vs 先捡神兵"
  "AP 1 用来拾取 vs 用来再走 1 格"
  → 战场内的微决策密度
```

### 5.3 战斗结束的自动结算

| 战斗结局 | 地面物品获得率 |
|---|---|
| **完全胜利**（全员存活 + 主目标达成）| 100% 全部地面物品自动获得 |
| **常规胜利**（主目标达成）| 100% 自动获得 |
| **撤退**（战略转移）| 30-50% 概率获得（看雇佣兵团是否有时间收）|
| **保护型任务**（护送 NPC）| 仅"必须保护对象身边 2 格"内的物品 |
| **失败 / 全军覆没** | 0%（所有人 + 所有物品丢失）|

→ **撤退是真实损失** —— 玩家学会"不能轻易撤"。

### 5.4 装备耐久与战利品

```
敌人死亡时装备耐久 = max(20%, 敌人剩余 HP 比例 × 100%)

例：
  敌人 HP 100/100 满血秒杀 → 装备 100% 耐久（神兵般完美）
  敌人 HP 30/100 → 装备 30% 耐久（破破烂烂的）
  敌人 HP 1/100 残血 → 装备 20% 耐久（最低保底）
  俘虏（活捉）→ 装备 100% 耐久 ⭐⭐

→ 玩家决策："这个戴神兵的精英我用瞄头一击秒 vs 慢慢消耗血"
  瞄头一击 → 神兵满耐久 ⭐
  慢慢消耗 → 神兵基本废了
  俘虏 → 神兵满耐久 + 不需要修理 ⭐⭐
```

→ **杀人方式选择影响装备质量**——这是 Berwick 的精髓。

### 5.5 与"贪财"性格的化学反应

> 见 `personality-system.md` §2.2.2。**不进战术 AI**。

```
贪财单位（敌我均可，有 personality_id）击杀敌人后：
  - 70% 触发：该次掉落数量 ×1.25~×1.5 或额外小袋金钱
  - BattleLog：【性格·贪财】××顺手揣走些许战利品
  - 音效：阴险低沉笑声（可设置关闭）
  - 友军触发：队友好感微降（周上限）

不做：AI 脱线拾取 / Loot Behavior（避免像 AI 弱智）
```

### 5.6 战利品稀有度

```
普通敌人（杂兵）：
  20% 掉落任意物品（金钱小袋 / 简陋武器）

精英敌人（带名字）：
  60% 掉落 1-2 件（武器 + 防具 + 金钱）

BOSS：
  100% 掉落（指定特殊战利品 + 普通战利品）

俘虏获得物品：
  100%（直接获取俘虏身上所有装备）+ 押送换额外金钱
```

→ "杀 vs 抓"经济差异**极大**——这就是俘虏经济的真实体现。

## 六、武器耐久 + 修理系统（学 Berwick）

### 4.1 耐久度机制

每件武器有耐久（Durability）：

| 武器档次 | 价格 | 初始耐久 |
|---|---|---|
| 普通货色（铁刀）| 100 | 30-50 次 |
| 良品（精铁陌刀）| 500 | 50-80 次 |
| 精品（鎏金陌刀）| 1500 | 80-120 次 |
| 神兵（传说级）| 5000+ | 150+ 次 |

**消耗**：
- 攻击命中：-1 耐久
- 攻击未命中：0 耐久（不消耗）
- 受到反击：-1 耐久（盾/重武器）

**耐久归零后**：
- 武器**损坏**（不能使用，但仍在背包）
- 必须送修才能恢复

### 4.2 修理系统

| 维度 | 规则 |
|---|---|
| **修理费用** | 武器价的 30%（普通）/ 40%（良品）/ 50%（精品）/ 60%（神兵）|
| **修理时间** | 普通 0 天 / 良品 1 天 / 精品 2 天 / 神兵 3 天（占用大地图时间）|
| **修理上限** | 神兵不能修过 5 次，第 6 次永久损坏 ⭐ |
| **修理地点** | 大城市铁匠铺 / 营地铁匠（要先招）|

→ **神兵不能无限修**——这是 Berwick 同款的"传家宝焦虑"。
→ 玩家心理："这把神兵只剩 2 次修理机会，今天的 BOSS 战用还是不用？"

### 4.3 装备焦虑机制

```
玩家每周 4-5 场战斗后的真实算账：
  - 4 把武器各打了 ~10 次 → 都还能用
  - 1 把神兵打了 5 次 → 还有 75% 耐久（修过 1 次）
  - 修理预算：总价 30%
  
玩家决策：
  ① 全修：花 800 金，下周打硬仗稳
  ② 只修 2 把神兵：花 300 金，省钱招新人
  ③ 全不修，下周用便宜货：省 800 金，但伤害不够
```

→ **每个决定都有真实代价**——这是 Berwick 的灵魂。

## 七、雇佣兵管理（学战兄弟）

### 5.1 招募系统

**招募来源**：
- 大地图城镇：每个城市有 1-3 个"待招募雇佣兵"，每周刷新
- 关卡奖励：某些关卡完成后解锁特定 NPC 招募
- 隐藏招募：完成隐藏条件触发（FE 同款攻略级招募）

**招募成本**：
```
新兵（Lv 1）：200-500 金（取决于初始属性）
有经验（Lv 5）：1000-2000 金
精英（Lv 10）：3000-5000 金
传奇 NPC：8000+ 金 / 或剧情触发
```

**随机生成**（学 XCOM）：
- 名字（唐朝姓氏库 + 名字库）
- 出身地（影响背景故事 + 初始 Trait 偏好）
- 初始属性（在范围内随机）
- **背景**（1 个，决定事件池）+ **性格**（1 个，决定事件选项）+ 初始 **Trait**（天赋，0～2 个）⭐；好感度初始 50，见 `camp-events-system.md`
- 体质天花板（隐藏属性，决定未来成长上限）⭐⭐

→ **没有两个一样的雇佣兵**——这是 XCOM/战兄弟的成瘾点。

### 5.2 永久死亡机制（学战兄弟）

```
雇佣兵血量归零 → 死亡 → 永久从队伍中移除
所有 EXP/JP 归零，装备保留（落在战场可拾取）

死亡后强制支出：
  - 家属抚恤金（100-500 金）
  - 团队士气下降（影响接下来 1 周战斗）

例外：
  - "急救" 主动技能可救濒死友军（医户/医官）
  - "还魂术"（医官 Lv3）可救刚死友军（一战 1 次）
  - 没有商店复活药 / 没有读档复活（玩家可手动读档，但游戏不鼓励）
```

→ **死亡是真死**——这就是雇佣兵 IP 的灵魂。
→ "保护我的明星雇佣兵" = 玩家的核心情感投入。

### 5.3 受伤系统（学 Berwick）

非死亡的"软死亡"：
```
雇佣兵血量归零但可能不死：
  - "倒地" 状态（30% 死亡概率，70% 重伤）
  
重伤后：
  - 缺席 1-3 周（在营地养伤，不能出战）
  - 期间需要支付医疗费（50 金/周）
  - 复出时可能附带"永久 debuff"
    （受伤部位决定：失去眼睛 → 命中 -10%, 失去手指 → 武器熟练 -5）
```

→ **死亡是终点，受伤是代价**——介于"全员存活"和"全军覆没"之间的中间态。

## 八、整周目经济节奏

### 6.1 玩家经济曲线（理想状态）

```
Phase 1（前 5 章）：温饱挣扎期
  - 收入：刚够工资 + 1 把修理
  - 玩家心态："我能撑过这周吗？"
  - 关键：必须接每个合同，必须省着用武器

Phase 2（6-9 章）：脆弱稳定期
  - 收入：能修武器 + 偶尔招新人
  - 玩家心态："这个明星雇佣兵我升不升？太贵了"
  - 关键：第一次有 build 选择空间

Phase 3（10-12 章）：精英队期
  - 收入：能购精品装备 + 团队完整
  - 玩家心态："最后一战要全员存活"
  - 关键：S 级条件成为重玩动机

Phase 4（13-15 章 + 终章）：传奇队期
  - 收入：富裕，但传说武器极贵
  - 玩家心态："拿真结局还需要哪 2 件神兵"
```

→ 经济曲线**永远紧绷**，但**渐进式宽松**——避免"前期窒息后期免疫"。

### 6.2 通关后的多周目经济（学 XCOM/Hades）

通关后解锁：
- **传家宝**：通关时存活的明星雇佣兵 → 二周目可"传承"装备/Trait
- **金钱倍率**：二周目+ 收入 -25%（增加难度）
- **新解锁内容**：上级职业 +2 / 新武器 / 新合同类型

→ **二周目不是"刷"，是"换打法"**——重玩动力来自系统而非数值。

| # | 决策 | 选定 |
|---|---|---|
| 1 | EXP 是否可 grind | ❌ 不能，固定池 |
| 2 | 武器是否有耐久 | ✅ 是 |

## 十二、风险与对策

| 风险 | 对策 |
|---|---|
| 玩家觉得"太穷玩不下去" | 教程关给 2 倍金钱 + 主线 1-3 章送 1 把神兵保底 |
| 玩家"省钱过头"导致后期打不动 | UI 提示"建议修理"+ 推送 S 级条件 |
| 武器耐久焦虑过载 | 普通武器修理 0 天 / 自动修（玩家选择"不去铁匠铺也修"）|
| 死亡惩罚太重导致"读档党" | 不做"Iron Man 强制"，但表彰"无死亡通关"奖杯 |
| 多周目枯燥 | 每周目解锁 2 上级职业 + 1 个传说武器系列 + 新事件 |
