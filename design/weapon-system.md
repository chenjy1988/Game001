# 武器系统 v2

> 本文档专门承载武器/装备相关的设计。design.md 主文档只保留索引指向本文档。
>
> 设计哲学：**FFT 风骨的简洁 + BB 派的减法制 + XCOM 派的随机词条 + 火焰纹章派的双向特技**。
> 核心目标：武器不抢人物培养风头，差异化来自"职业 × 武器 × 词条 × 持有者状态"的组合，而非武器数值堆料。

---

## 一、设计哲学（不可摇摆）

### 5 条核心原则

1. **武器属性极简**——普通武器只有 7 字段，比 FFT/TO 还简洁
2. **取消武器伤害上下限**——伤害基础值单一数字，浮动来自气力档位（玩家可预判）
3. **差异化来自人，不来自武器数值堆料**——同一把武器在不同职业/状态下表现完全不同
4. **传说武器不一定绝对强**——"上限更高、特化更极端"，但需要适合的角色才发挥威力，不抢人物培养
5. **词条双向**——传说级词条几乎都有"得有失"，借鉴火焰纹章特技，让玩家根据角色取舍

### 反极端爆发红线（与 design.md 战斗模型 v3 第七节同步）

> 设计原则更新：取消"硬性 cap"，改为**互斥规则 + 触发概率 + 词条上限**三重自然约束（详见 4.8 节）。极端组合（如全力一击 + 头部 + 暴击）仍可触发，但概率 < 2%，作为"破甲爆发瞬间"的设计回报。

- **职业能力互斥**：一次攻击只能选 1 个职业能力（普攻/瞄头/全力/精准/穿心刺/碎甲/...）
- **加法叠加**：头部 +0.5 + 暴击 +0.5 = ×2.0 上限（不超）
- **词条加成总和 ≤ +30%** 武器伤害（红线）
- **职业被动总和 ≤ +30%**（红线，针对"基础被动"如狂暴/快射等）
- **条件触发型被动可超出 +30% 红线**（如锐意/残忍/背刺等需满足触发条件的被动）——但需保持互斥规则：
  * 残忍 vs 背刺：互斥（同角色二选一）
  * 触发条件天然限制频率（侧后方需走位、有伤势需队友配合、满血需未受伤）
- **武器升级路径（普通 → 传说）至多 +30% 数值**（特效不计入此上限，但单个特效遵守红线）

## 二、三层架构总览

```
第一层：基础武器属性（精简，所有武器通用）
  └─ 7 个核心字段，决定基线表现
       ↓
第二层：职业武器专精（差异化核心）
  └─ "X 职业 + Y 武器" → 更少 AP + 更高威力 + 解锁风格
       ↓
第三层：传说级随机词条（XCOM 派 + 火纹双向特技）
  └─ 顶级武器随机词条，确保 build 探索性 / 词条几乎都双向
```

## 三、第一层：基础武器属性（精简到 7 字段）

### 3.1 通用字段

| 字段 | 类型 | 例：横刀 | 例：擘张弩 | 说明 |
|---|---|---|---|---|
| `id` | string | "saber" | "crossbow" | 唯一标识 |
| `name` | string | "横刀" | "擘张弩" | UI 显示 |
| `attack_modes` | list | ["slash"] | ["pierce"] | 攻击类型列表（slash/pierce/crush），多模式武器有 2 个选项 |
| `damage_base` | int | 42 | 75 | **单一基础伤害**（替代 min/max）|
| `ap_cost` | int | 4 | 6 | 攻击 AP |
| `range` | [int, int] | [1,1] | [2,6] | [最小, 最大] 射程 |
| `block_value` | int | 8 | 0 | 格挡概率（独立检定）|
| `weight` | int | 4 | 8 | 装备重量（叠加到角色 weight，影响 stamina weight_mult）|

### 3.2 远程武器特殊字段（按需）

| 字段 | 类型 | 例 | 说明 |
|---|---|---|---|
| `reload_ap` | int | 3 | 弩专属：发射后必须消耗的上弦 AP |

### 3.3 全局规则（不写在武器表里）

- **暴击伤害**：×1.5（统一）
- **暴头伤害**：×1.5（统一）
- **微小波动**：所有伤害 × random(0.95, 1.05)（用力大小的微差，不影响策略）
- **气力档位系数**：见第六节
- **攻击类型 → 伤害分配（Slash/Pierce/Crush 的 armor_mult、hp_mult、渗透系数）**：见第四节
- **普攻 / 瞄头 / 全力一击 / 精准射击**：通用职业能力，受武器物理限制（详见 4.5 节）
- **专属职业能力**（穿心刺 / 横扫千军 / 剑舞 / 抵近射击 / 碎甲 / 击晕 等）：由"职业 × 武器组合 + JP 投资"解锁（详见 4.9 节）

## 四、攻击类型与穿透机制（三分法 + 武器多模式）

### 4.1 设计哲学：穿透 = 经济最大化的工具

伤害穿透不是数值优势，而是**给玩家一条高风险高回报的选项**：

- **稳健路径**（劈砍/钝击）：先破甲再杀人 → 安全但慢、消耗多 AP/气力
- **冒险路径**（穿刺找甲缝）：跳过破甲直接致命 → 快但难命中、有反伤风险

→ 玩家根据角色 build 和当前局势选择路径，**穿透 = 玩家技术的回报 + 战略选择的工具**，与 Berwick 经济紧缺哲学深度契合。

### 4.2 三种攻击类型 + 武器重量驱动的渗透系数

护甲不再分材质（参考 design.md 单维度 weight）。攻击类型决定破甲速度与 HP 输出特性，**渗透系数由攻击类型基础率 × 武器重量修饰共同决定**——重型利器的"动能"让钝伤透甲、戳穿心脏、人马俱碎成为可能。

#### 4.2.1 攻击类型基础参数

| 攻击类型 | armor_mult（破甲）| hp_mult（破甲后）| **基础渗透率** | 命中修正 | 物理直觉 |
|---|---|---|---|---|---|
| **Slash 劈** | 1.0 | 1.0 | **0.10** | 0 | 砍 → 锋利切入 + 钝面冲击，少量内伤 |
| **Pierce 穿刺** | 0.7 | 1.0 | **0.15** | -5% | 戳 → 找甲缝戳进去，戳穿要害 |
| **Crush 砸** | 1.5 | 0.7 | **0.08** | 0 | 击 → 钝头碎甲传力，钝伤透甲 |

**关键设计变化**（v2 修订）：
- ❌ 旧：仅 Pierce 有 0.15 渗透，Slash/Crush 为 0（"砍/砸不破甲就 0 内伤"，反直觉）
- ✅ 新：三种攻击类型都有破甲前渗透（锋利度梯度：Pierce 0.15 > Slash 0.10 > Crush 0.08）
- ✅ 新：渗透系数 × **武器重量修饰**（重型利器渗透更强，匹配"陌刀人马俱碎"的物理直觉）

#### 4.2.2 武器重量修饰公式 ⭐

```
weight_modifier = 1.0 + max(0, (weight - 4) / 6)

→ weight ≤ 4：modifier = 1.0（基线，匕首/弓/横刀等轻量武器）
→ weight 6（短矛/长棍）：modifier = 1.33
→ weight 8（单手战锤）：modifier = 1.67
→ weight 12（长矛/弩）：modifier = 2.33
→ weight 14（双手战斧/陌刀）：modifier = 2.67
→ weight 16（双手战锤）：modifier = 3.0
```

**实际渗透 = 基础渗透率 × weight_modifier**

#### 4.2.3 全武器实际渗透矩阵

| 武器 | weight | 类型 | 基础率 | modifier | **实际渗透** |
|---|---|---|---|---|---|
| 匕首 | 0 | Pierce | 0.15 | 1.0 | **0.15** |
| 短矛 | 6 | Pierce | 0.15 | 1.33 | **0.20** |
| 横刀 | 6 | Slash | 0.10 | 1.33 | **0.133** |
| 剑 (Pierce) | 6 | Pierce | 0.15 | 1.33 | **0.20** |
| 单手战锤 | 8 | Crush | 0.08 | 1.67 | **0.133** |
| 单手战斧 | 10 | Slash | 0.10 | 2.0 | **0.20** |
| 步弓 | 4 | Pierce | 0.15 | 1.0 | **0.15** |
| 复合弓 | 5 | Pierce | 0.15 | 1.17 | **0.175** |
| 长棍/禅杖 | 6 | Crush | 0.08 | 1.33 | **0.107** |
| 长矛 | 12 | Pierce | 0.15 | 2.33 | **0.35** ⭐ |
| 双手战斧 | 14 | Slash | 0.10 | 2.67 | **0.267** ⭐ |
| 双手战锤 | 16 | Crush | 0.08 | 3.0 | **0.24** ⭐ |
| 陌刀 (Slash) | 14 | Slash | 0.10 | 2.67 | **0.267** ⭐ |
| 陌刀 (Pierce) | 14 | Pierce | 0.15 | 2.67 | **0.40** ⭐⭐ |
| 擘张弩 | 12 | Pierce | 0.15 | 2.33 | **0.35** ⭐⭐ |

→ **重型利器（长矛/陌刀/双手战斧/擘张弩）渗透 0.20-0.40，戳穿/砍透/钝击透甲都成立**
→ **轻量武器（匕首/弓/单手剑）渗透维持基础率**，靠职业能力（穿心刺/暗杀）补足

#### 4.2.4 伤害分配规则

```
if Body Armor > 0（甲未破）：
  Body 扣 = damage × armor_mult
  HP 扣 = damage × 实际渗透  // 实际渗透 = 基础率 × weight_modifier

if Body Armor = 0（已破甲）：
  HP 扣 = damage × hp_mult
```

**Pierce 的 -5% 命中惩罚**：找甲缝戳进去需要精度，是物理代价，也是平衡设计（防止 Pierce 普攻超模）。

**心智负担**：玩家不需要算公式——每件武器的实际渗透值直接显示在武器卡上。设计师按 § 4.2.3 表配新武器即可。

### 4.3 武器多模式攻击（剑/陌刀双模式）

部分武器物理上支持多种攻击类型，玩家在攻击菜单临场选择：

| 武器 | 主攻击 | 副攻击 | 设计理由 |
|---|---|---|---|
| 匕首 | Pierce | — | 太轻只能戳 |
| 横刀 | Slash | — | 单刃直刀，劈为主 |
| **剑** | Slash | **Pierce** ⭐ | 双刃多用，玩家选择劈或刺 |
| 短矛 | Pierce | — | 短矛单手戳 |
| 长矛 | Pierce | — | 双手长杆只能戳 |
| 战斧（单/双手）| Slash | — | 重斧专劈 |
| **陌刀** | Slash | **Pierce** ⭐ | 长柄双刃，剑刺也能 |
| 战锤（单/双手）| Crush | — | 钝头专击 |
| 长棍/禅杖 | Crush | — | 钝面 |
| 步弓/复合弓 | Pierce | — | 箭头戳 |
| 擘张弩 | Pierce | — | 弩箭戳 |

**UI 设计**：
- 攻击菜单显示当前武器的所有可用攻击类型
- 每种类型显示伤害预览：`劈：40 Body / 0 HP` vs `刺：28 Body / 6 HP`
- 玩家用结果做决策，不需要懂内部公式

### 4.4 攻击类型的命中修正（与匕首特殊性的统一）

Pierce 武器普攻基础 -5% 命中是**找甲缝的物理代价**。匕首作为单手 Pierce 武器，**普攻命中无额外惩罚**（已被 damage_base 28 + 无全力一击 + 不能装重甲等多重机制约束，不需要额外普攻惩罚）。

| 武器分类 | 普攻命中修正 | 备注 |
|---|---|---|
| Slash 武器（横刀/剑劈/战斧）| 0 | 基线 |
| **Pierce 武器**（匕首/短矛/长矛/剑刺/弓/弩）| **-5%** | 找甲缝代价（统一） |
| Crush 武器（战锤/长棍）| 0 | 基线 |

→ 反极端爆发约束体现在"瞄头惩罚 + 互斥规则 + 触发概率 + 词条上限"四重自然约束（详见 § 一末尾的红线 + 4.8 节），而不是普攻命中惩罚。

### 4.5 通用职业能力（受武器物理限制）

**核心定位调整**：所有攻击行为都是**职业能力**，不是武器属性。武器只决定"物理上能不能做"，能不能学到、什么时候能用，由职业决定（职业系统详见 class-system.md）。

按"是否所有职业默认会"分两类：
- **通用职业能力**（本节）：普攻 / 瞄头 / 精准射击 / 全力一击——大多数职业默认会，受武器物理限制
- **专属职业能力**（4.9 节）：穿心刺 / 横扫千军 / 剑舞 / 抵近射击 / 碎甲 / 击晕 ……——通过 JP 投资，"职业 × 武器组合"专属解锁

#### 普攻（默认）
- AP 与武器基础一致
- 命中后做"部位检定"：
  - **单手武器**：25% 概率打头部（×1.5 伤害）——BB 派核心机制
  - **双手武器**：**15% 概率打头部**——重武器挥砍轨迹大、惯性强，精准打头更难
- 25% 头部命中是 BB 派核心机制，双手武器降到 15% 是物理代价
- **15% vs 25% 的量化影响**：双手武器期望 multiplier 从 1.125 降至 1.075（-4.4% 平均输出），头部爆发触发从每 ~5 刀 1 次降至每 ~9 刀 1 次

#### 精准攻击（单手近战 + 远程专属，AP 攻+2）

统一公式：
- 命中 **+15**（精准优于普攻但不无脑——打脆皮普攻更优，打高防精准更优）
- 气力消耗 **base 8**（> 2×普攻 6，不能无脑用）
- 头部概率 25% → **55%**（远程 50%）
- 界面显示名：Slash = 斩首 / Pierce = 爆头 / 远程 = 瞄头
- **双手武器不能使用精准攻击**（重武器挥砍轨迹大，无法精准戳要害）

#### 全力一击（双手近战武器专属）
- AP +1
- 隐藏气力消耗 +12
- 命中 **-10%**
- 伤害 **+15%**
- **本回合自身近战防御 -10**（挥砍轨迹大、露出破绽——双向特技代价）
- 头部概率仍按双手武器基线 15%，不提升
- **唯一爆发模式**：双手武器没有精准攻击，全力一击是它的专属爆发路径

#### 武器物理限制总览

每种"通用职业能力"由武器物理性质决定能不能用：

| 能力 | 单手近战 | 双手近战 | 远程 |
|---|---|---|---|
| 普攻 | ✅ | ✅ | ✅ |
| 精准攻击（斩首/爆头/瞄头）| ✅ | ❌ | ✅ |
| 全力一击 | ❌ | ✅ | ❌ |

→ 两个爆发模式各对应武器物理形态，互不替代：
- 单手+远程专属爆发 = **精准攻击**（命中 +15 / 头部 55% / 无伤害加成）
- 双手专属爆发 = **全力一击**（伤害 +15% / 头部仍 15% / 防御 -10）

**互斥规则统一处理**：见 4.8 节——所有职业能力（通用 + 专属）一次攻击只能选 1 个。

### 4.6 精准攻击（已统一）

> ⚠️ 精准攻击已统一为命中 +15、头部 55%（近战）/ 50%（远程）。不再有武器间命中惩罚差异。详见 §4.5。旧瞄头惩罚表已废弃。

注：长矛 / 双手战斧 / 双手战锤 / 陌刀 / 长棍/禅杖 都是双手武器，**不能瞄头**。

### 4.7 暴击与头部命中的叠加（加法）

**关键规则**：暴击和头部命中都是 ×0.5 加成，**加法叠加**而非乘法：

```
multiplier = 1.0
if 头部命中：multiplier += 0.5  → ×1.5
if 暴击：multiplier += 0.5      → ×1.5
两者同时：multiplier = 2.0     → ×2.0

最终 = base_damage × multiplier
（不再有 cap 截断——见 4.8 节互斥+概率自然约束）
```

**为什么用加法而非乘法**：
- 乘法叠加 ×1.5 × 1.5 = ×2.25，远高于加法 2.0
- 加法叠加更线性、玩家心智更简单：每个 0.5 加成是独立来源
- 同时触发"头部+暴击"概率天然低（双手 15% × 5% = 0.75%；单手 25% × 5% = 1.25%），自然成为"爆发瞬间"

**暴击的攻击类型差异化**（在 ×1.5 基础上的额外效果）：

| 攻击类型 | 暴击额外效果 | 暴击基础渗透率（受 weight_modifier 加成）|
|---|---|---|
| Slash 暴击 | ×1.5（基线，无额外效果）| **0.20**（基础率 0.10 翻倍）|
| Pierce 暴击 | ×1.5（"刺中要害"）| **0.30**（基础率 0.15 翻倍）|
| Crush 暴击 | ×1.5 + 直接破甲 30 点（"震碎甲叶"）| **0.16**（基础率 0.08 翻倍）|

**实际暴击渗透 = 暴击基础率 × weight_modifier**（weight 修饰公式见 § 4.2.2）

**举例**：陌刀（Pierce 模式 weight 14）暴击渗透 = 0.30 × 2.67 = **0.80** ⭐
→ 陌刀 Pierce 暴击命中：综合伤害 ×0.80 直接转 HP，配合"全力一击 + 头部 + 暴击"瞬间 = 一击斩重甲。

#### 暴击率（crit_chance）公式

```
crit_chance = 5（基础）
            + 武器暴击加成（词条："锋锐" +5、"致死" +10 等）
            + 职业精通加成（精通武器 +5 暴击）
            + max(0, wisdom - 40) × 0.2     ← Wisdom 加成 ⭐
            + 通用天赋 JP 投资

软上限 50%（神装 + 高 Wisdom + 高 JP 投资才能接近，不硬截断）
```

**Wisdom 暴击加成的设计意图**（详见 class-system.md § 4.3）：

| Wisdom | 暴击加成 | 受益职业 |
|---|---|---|
| ≤ 40 | 0 | 重武器职业（陌刀手、悍卒、押衙等天然 Wisdom 低）不亏 |
| 50 | +2 | 中等加成 |
| 80 | +8 | 斥候 / 不良人 / 诗剑仙 / 方士战兵等 Wisdom 系职业基线 |
| 100 | +12 | 神装 + 满养成顶配 |

→ Wisdom 系 + 高速 build 的核心爆发是**多次独立暴击 roll**（双武器精通、高 Init 多回合行动），暴击率提升 8-12% 在长线对战中显著放大爆发期望。投资 Wisdom = 投资暴击 build。

### 4.8 职业能力互斥规则（统一）

**核心规则：一次攻击只能使用 1 个职业能力（通用 + 专属合并管理）**

通用职业能力（4.5 节）：
- 普攻 / 瞄头 / 精准射击 / 全力一击

专属职业能力（4.9 节）：
- 穿心刺 / 横扫千军 / 剑舞 / 抵近射击 / 碎甲 / 击晕 / 重击 / 锐不可当 ……

**互斥规则**：
- 一次攻击只能选 **1 个**——比如选了"全力一击"就不能再叠"碎甲"
- 玩家根据局势取舍：要破甲选碎甲、要爆发选全力、要稳定选普攻

**被动效果可叠加**（不在互斥范围内）：
- 职业被动（如不良人背刺加成）：满足条件自动触发
- 词条被动（如渴血、千钧一发）：自动触发
- 武器双手握持加成（5.5 节）：装备配置的物理后果，自动触发

**自然约束（替代过往的硬 cap）**：
1. **互斥规则**——伤害乘法源最多 1 个（来自所选职业能力）
2. **加法叠加**——头部+暴击 ×2.0 已是加法上限
3. **概率封顶**——双手 15%×5% = 0.75% 极端事件率，单手 25%×5% = 1.25%
4. **词条总和 ≤ +30%**（红线）
5. **职业被动总和 ≤ +30%**（红线）

→ 极端组合的伤害不再被 cap 截断，但出现概率 < 2%——这是"破甲爆发瞬间"的设计意图，玩家几小时打不出 1 次，打出来时尖叫。

### 4.9 专属职业能力（武器回归本质）

**设计哲学（核心）**：
> **武器是工具，人才决定能用什么招**。武器只决定基础数据 + 物理属性（攻击类型、单/双手、是否远程），不内置任何能力。所有能力（通用 + 专属）都是**职业的能力**——通用类多数职业默认会，专属类需要 **职业 × 武器组合 + JP 投资**解锁。

#### 4.9.1 武器层 vs 通用职业能力 vs 专属职业能力

```
武器层（物理工具）：
  damage_base / AP / range / block / weight
  攻击类型（Slash/Pierce/Crush）
  持握类型（单手/双手/远程）
  → 武器不决定能用什么招，只决定物理可能性

通用职业能力层（4.5 节，多数职业默认，受武器物理限制）：
  普攻 / 瞄头 / 精准射击 / 全力一击
  → 哪些职业默认会、哪些职业被禁用，由职业系统决定（class-system.md）
  → 受武器物理限制（瞄头需单手、全力需双手等）

专属职业能力层（本节，职业 × 武器组合 + JP 投资）：
  穿心刺、横扫千军、剑舞、抵近射击、骑乘冲锋、碎甲、击晕、重击……
  → 通过 JP 投资学习，是职业身份的核心载体
  → 与通用类共用 4.8 节的"一次攻击只能选 1 个"互斥规则
```

#### 4.9.2 职业技能：穿心刺 Puncture（详细演示）

**解锁条件**：
- **可学职业**：斥候 / 游侠 / 不良人 / 诗剑仙 / 任何使用匕首的职业
- **武器要求**：必须装备匕首
- **JP 等级**：初级（任何阶段都能学，不是高阶）
- **不良人天赋加成**：穿心刺暴击率 +5、命中惩罚 -5%（专精）

**基础规则**：

| 项 | 数值 |
|---|---|
| AP 消耗 | base + 1（普通匕首 3 → Puncture 4 AP）|
| 隐藏气力消耗 | **+12**（与 BB Puncture 20 锚点对齐，营造"用一次想三次"节奏） |
| 命中修正 | -15%（在 Pierce 普攻 -5% 基础上叠加 = 总 **-20%**） |
| 护甲处理 | **完全无视**（不扣 Body / Head Armor） |
| 伤害去向 | 100% 转 HP |
| 部位 | **强制命中身体**（不掷头部检定，不享受瞄头加成） |
| 暴击 | **可暴击**（但暴击率自然低 = 基础 5% + 武器/天赋加成） |
| 互斥 | 占用"职业能力"槽，本次攻击不能再叠其他能力（4.8 节）|

**伤害公式**：

```
最终伤害 = damage_base × 气力档位 × 微波 × 专精系数 × (1 + 词条加成)
       × 暴击修正（可触发，按 5%~30% 概率）

→ 不参与"头部 +0.5"（默认身体）
→ 不与瞄头叠加（占用"职业能力"槽，互斥）
→ 完全无视护甲，伤害 100% 转 HP
```

**关键设计选择**（不同于 BB 派的硬限制）：
- ❌ BB：硬性"不可暴击"
- ✅ 我们：**允许暴击**，暴击率天然低（基础 5% + 武器/天赋加成）
- ❌ BB：硬性"不可瞄头"
- ✅ 我们：**默认强制身体**，传说级"心眼"词条可解锁瞄头穿刺（Phase 4）
- → 用数值平衡而非机制硬限制，玩家培养空间更大

**演示：穿心刺成长曲线**

#### 前期斥候/游侠（普通匕首 + 初学穿心刺）
- 装备：普通匕首（damage_base 28，无词条）
- 配置：合格匕首（专精 1.0），暴击率 5%
- Puncture 数值：
  - 期望伤害：28 × 1.0 = **28 HP/击**
  - 命中率：75% - 20% = **55%**
  - 暴击伤害（5% 概率）：28 × 1.5 = **42 HP/击**
- 击杀演算（HP 100 目标）：
  - 期望击杀次数 = 100 / (0.55 × 28) = **6.5 击**
- **定位**：保甲斩杀的功能型工具，节奏慢但保留装备完整

#### 后期不良人（传说匕首 + JP + 精通）
- 装备：传说级匕首"穿心匕首" + 词条"致死"（+暴击 10）+"锋锐"（+暴击 5）+"利刃"（+5%伤）
- 配置：不良人精通匕首（专精 1.2）+ JP 投入暴击天赋 +5 + 不良人 Puncture 专精 -5% 命中惩罚
- 暴击率：5 + 5（不良人天赋）+ 10 + 5 + 5（JP）= **30%**
- Puncture 数值：
  - 基础伤害：28 × 1.2 × 1.05 = **35.3**
  - 暴击伤害：35.3 × 1.5 = 53.0
  - 命中率：75 - 20 + 5（不良人专精）= **60%**
- 击杀演算（重甲 HP 130 目标）：
  - 期望伤害每击 = 0.6 × (0.3 × 53 + 0.7 × 35.3) = **24.4 HP/击**
  - 击杀次数 = 130 / 24.4 = **~5.3 击**
- **定位**：重甲精英杀手 + **战利品完整保留**（Body Armor 130 / Head Armor 80 都不损）

#### Phase 4 终极 build（传说级"心眼"词条解锁瞄头穿刺）
- 词条"心眼"：解锁穿刺时可瞄头（命中再 -15%）
- 极致爆发：35.3 × 1.1（瞄头）× 2.0（头部+暴击）= **77.7 HP**
- 命中率：60% - 15% = **45%**
- 暴击 + 头部触发：30% × 70% = 21%
- → 一击 77.7 HP 的概率约 **9.5%**，护甲完整保留，是终极保甲斩杀
- → 这是"破甲爆发瞬间"的设计意图，9.5% 出 1 次的尖叫感正是 Pierce 路线的回报

**对比表：穿心刺 3 阶段战术地位**

| 阶段 | 期望伤害/击 | 单击爆发上限 | 触发概率 | 战术地位 |
|---|---|---|---|---|
| 前期（普通匕首 + 初学）| ~15 HP | 42 HP（暴击）| 5% | 保甲功能型 |
| 后期（传说匕首 + 不良人精通）| ~24 HP | 53 HP（暴击）| 30% | 重甲杀手 |

### 5.4 武器差异化注释

**剑 vs 横刀**（同 4 AP）：
- 横刀：Slash 单模式，damage 42 / block 8 / 瞄头 -25% → 稳定砍劈
- 剑：Slash + **Pierce 双模式**，damage 40 / block 10 / 瞄头 -20%（劈）/ -15%（刺）→ 灵活切换

**陌刀双模式（weight 14 → 渗透 modifier 2.67）**：
- Slash 模式：75 × 1.0 = 75 Body / 75 × **0.267** = **20.0 HP**（破甲前）→ 快速破甲 + 钝伤透甲，"人马俱碎" ⭐
- Pierce 模式：75 × 0.7 = 52.5 Body / 75 × **0.40** = **30.0 HP**（破甲前）→ 戳穿要害 ⭐⭐
- 暴击瞬间（Pierce + 暴击）：渗透 ×2 = 0.80 → 75 × 0.80 = **60 HP 直伤**，配合"全力一击 + 头部 + 暴击"= 一击斩重甲

**单手 vs 双手版本（战斧/战锤）**：
- 单手：伤害低、AP 低、留副手位（盾或副武器）、能瞄头
- 双手：伤害高、AP 高、占满双手、不能瞄头、解锁全力一击

**双手战锤 vs 长棍/禅杖（同 Crush 双手类，分工不同）**：
- 双手战锤：6 AP / damage 70 / block 0 / weight 8 → **一击重击型**，单回合 1 击 = 70 base
- 长棍/禅杖：4 AP / damage 35 / block 12 / weight 5 → **高频持续型**，单回合 2 击 = 70 base，但能格挡
- 两者单回合产出基线相当，但**爆发分布、防御能力、AP 节奏**完全不同；不需要合并

**盾**：装备到副手，提供 +25 block_value（独立装备，不在武器表）。

**双武器精通**：装备 2 把单手轻型武器（匕首/横刀/剑/短矛），需要"双武器精通" Trait 才有加成（副手 +15 block + 解锁双攻技能）。

### 5.5 双手握持（Double Grip）——单手武器空副手时 +25% 伤害（已锁定）

**物理含义**：单手武器空副手时，辅助手握住刀柄/枪杆提供力矩加成，砍/戳得更狠。陌刀等双手武器本身就是双手握持，不存在"单持"概念——此机制仅适用于单手武器。

**最终值**：**+25% 伤害**（与 BB 原版锚点对齐）。

**为什么锁定 +25%**：调整后 2h/1h 平均基础伤害比 ≈ 1.66×（横刀 42 → 双手战斧 80 / 陌刀 75 / 双手战锤 70 / 长矛 55），DG 后 2h 仍高出 1h+DG 约 33%，满足"双手武器伤害优势必须保持"的设计约束（详见 5.6 节）。

#### 副手四选一格局

| 配置 | 副手 | 输出 | 防御 | 灵活度 | Weight | 定位 |
|---|---|---|---|---|---|---|
| 盾+单手 | 盾（+25 block）| 基线 | 高（block ~33%）| 中 | +2-3 | 防御型 |
| 双持 | 副武器（+15 block + 双攻）| 基线 + 双攻 | 中（block ~23%）| 高 | +1-4 | 技巧型 |
| **双手握持** | 空（Double Grip +25% 伤害）| **基线 +25%** | 低（仅武器 block ~8-10%）| 高 | 最低 | **玻璃大炮** |
| 远程+单手 | 步弓/弩 | 远程输出 | 低 | 中 | +3-8 | 远程型 |

#### 数值验证（精通，普攻；陌刀按调整后 base 75 计算）

| 配置（以横刀为例）| effective base | 期望伤害/击 | AP/击 | 伤害/AP | block | 核心权衡 |
|---|---|---|---|---|---|---|
| 盾+横刀 | 42 × 1.2 = 50.4 | 50.4 | 4 | 12.6 | ~33% | 基线防御 |
| 双持横刀 | 50.4 | 50.4 | 4 | 12.6 | ~23% | 双攻技巧 |
| **双手握横刀** | 50.4 × 1.25 = **63.0** | **63.0** | 4 | **15.75** | ~8% | +25% 伤害 / -25% block |
| 陌刀（双手武器）| 75 × 1.2 = **90** | **90** | 7 | 12.86 | ~15% | 全力一击 90×1.15=103.5 |

#### 双手握持 vs 双手武器的关键差异

1. **全力一击是双手武器独有爆发**——双手握横刀最高单击（普攻 + 头部 + 暴击）= 63.0 × 2.0 = 126；陌刀全力一击 + 头部 + 暴击 = 75 × 1.2 × 1.15 × 2.0 = **207**。双手武器的不可替代价值在 **全力一击的爆发上限**，不在普攻。

2. **AP 效率合理分化**——横刀双手握 AP 效率 15.75 vs 陌刀 12.86（+22%），但单回合产出：横刀 2 击（8 AP）= 126 vs 陌刀 1 击（7 AP）= 90，剩 2 AP 走不了第二击。**单手 = 高频稳定，双手 = 单击爆发**——两条路线，互不替代。

3. **极端组合靠互斥+概率自然约束**（4.8 节）——双手握横刀普攻头部+暴击 = 126，触发概率 25% × 5% = 1.25%；陌刀全力一击 + 头部 + 暴击 = 207，触发概率 15% × 5% = 0.75%。极端值出现频率 < 2%，是"破甲爆发瞬间"的设计意图。

4. **防御代价是真实代价**——盾牌 block ~33% → 空手 block ~8%，少 ~25% 格挡率。在减法制体系下，少 25% block ≈ 每回合多挨一次完整伤害，这是玻璃大炮应有的风险。

#### 加成来源约束检查

- 双手握持 ×1.25 是装备配置的物理后果，不计入"词条加成总和 ≤ +30%"
- 双手握持算 1 个乘法来源，与头部/暴击加法叠加；与全力一击/职业能力共存（DG 是装备状态，不占职业能力槽）
- 极端组合：双手握横刀 + 全力一击 ❌ 不存在（全力一击仅双手武器；DG 仅单手武器）—— **天然互斥**
- 单手武器最大单击：base × 1.25（DG）× 1.2（精通）× 2.0（头部+暴击）× 1.30（词条上限）= **base × 3.9**，触发概率 < 0.1%，作为"奇迹瞬间"接受

#### 5.6.2 BB Reforged 的核心判断

Reforged 开发团队的明确结论：**"单手武器不应该走单纯堆伤害来与双手武器持平这种无聊路线。"** 他们的设计方向：
- 双手武器 = 高伤害 + Reach Advantage（触及优势）
- 单手武器 = 低伤害但更通用，核心优势在**可以搭配盾牌**

**BB 社区共识**：Double Grip +25% 不够让 1h 追上 2h，但这是有意为之——2h 本来就应该比 1h 伤害高。

#### 5.6.3 我们的最终对齐结果

| 设计原则 | BB 锚点 | 我们的实现 |
|---|---|---|
| 2h/1h 基础伤害比保持 1.5×~2× | 1.75× | **1.66×**（横刀 42 → 双手系平均 70）✅ |
| DG +25%（不追平 2h）| +25% | **+25%** ✅ |
| 全力一击作为双手专属爆发 | +20%（Overhead）| **+15%**（更克制，配双向特技 -10 防御）✅ |
| 1h 高频 vs 2h 爆发分化 | 同 AP | 我们 1h(3-4 AP) / 2h(5-7 AP) **更激进**，分化更鲜明 ✅ |
| 无 hard cap，靠互斥+概率约束 | BB 也无 cap | ✅ 与 BB 一致 |
| 反盾/控场/破甲 = 武器自带 | 武器自带 | **职业 × 武器组合 + JP 解锁**（更深度，差异 1）|

→ 当前武器系统已经吸收 BB 体系的核心法则，并通过"职业能力体系"和"Attack Type 三分法"实现了我们独有的简洁化。

### 5.7 副手装备规则（2h2w）⭐

**核心哲学**：贴合真实世界——任何人都能左右手各持一把单手武器。这不是技能，是装备状态。

#### 5.7.1 副手四选一格局（任何人，无前提）

| 配置 | 副手 | 提供 | 副手能否攻击 |
|---|---|---|---|
| 盾+单手 | 盾 | +25 block_value | — |
| **2h2w 默认** | 任意单手武器 | **副手 block_value × 0.7**（向下取整）| ❌ 不能 |
| 双手握持 | 空 | 主手 +25% 伤（5.5 节）| — |
| 远程+单手 | 步弓/弩 | 远程切换 | 远程攻击 |

**默认 2h2w 规则**（任何人）：
- 任意单手武器都能放副手（不限 weight）
- 副手 block 加成 = 副手武器 block_value × 0.7（向下取整）
  - 副横刀（block 8）→ 副手 block +5
  - 副剑（block 10）→ 副手 block +7
  - 副匕首（block 3）→ 副手 block +2
  - 副短矛（block 6）→ 副手 block +4
- 副手不参与攻击 ❌
- 副手词条不生效 ❌
- 副手只是"装备状态" → 真实世界中的左右开弓，但不学会用副手就只是摆设

#### 5.7.2 双武器精通（专属能力，刺客 4 职业可学）

```
双武器精通：
  解锁职业：游侠 / 不良人 / 奇兵 / 斥候
  装备要求：副手单手武器（任意 weight）
  JP 等级：中级

  效果：
    1. 副手词条生效（双手词条独立作用，不交叉）
    2. 主手任何攻击命中后，副手追加 1 次普通攻击：
       - 副手伤害 = 副手 damage_base × 0.75 × 精通 × (1+词条加成)
       - 独立 roll 暴击/部位/命中
       - 不消耗额外 AP
       - 仅触发 1 次（不论主手打了几下，含疾风连击 3 段也只触发 1 次副手追击）
       - 主手 MISS 时，副手不追击
    3. 副手攻击始终是"普通攻击"形态，不复制主手的特殊效果
       （瞄头 / 穿心刺 / 放血等都不传染给副手）
```

**适用范围**（主手命中触发副手追击的场景）：
- ✅ 普攻 → 副手追击 1 次
- ✅ 瞄头 → 副手追击 1 次（普通攻击形态，不享受瞄头加成）
- ✅ 穿心刺 → 副手追击 1 次（副手是普通 Pierce 攻击，不是穿心刺）
- ✅ 疾风连击 3 段 → 3 段全部结束后副手追击 1 次（不是每段都触发）
- ✅ 放血 / 涂毒 → 副手追击 1 次（副手不附加流血/毒）
- ❌ 主手 MISS → 副手不动

#### 5.7.3 双武器精通的战术体感

**情境 A：神装不良人 + 双匕首 + 学过双武器精通 + 锐意 + Wisdom 80**：
- 主手暴击触顶 50%，副手暴击 43%
- 第 1 回合 9 AP（疾风 5 + 普攻 3 + 走位 1）：
  - 主手 4 击 + 副手 2 次追击 = 6 次结算
  - 期望伤害 ~210 综合（90% 命中下）
  - 打 Body 80 / HP 80：HP 总损失 ~98（秒杀概率 ~80%）
- → 神装稳秒低防中甲后排（刺客本职）

**情境 B：新兵 + 横刀+横刀（不学专精）**：
- 副手仅提供 block 5（不参与攻击）
- 单回合 2 次主手普攻 = 2 次结算
- 体感：双持纯防御选项，无输出加成

→ 双持的"输出收益"严格绑定专属能力，**不学专精双持就是装饰**——符合"少部分职业才有"的设计意图。

## 六、伤害公式（damage_base + 气力档位 + 互斥规则）

### 6.1 完整战斗流程

```
Step 1：选择职业能力（一次攻击只能选 1 个，4.8 节互斥规则）
  通用职业能力（4.5 节，受武器物理限制）：
    - 普攻（默认）
    - 精准攻击：斩首（单手 Slash）/ 爆头（单手 Pierce）/ 瞄头（远程）
    - 全力一击（仅双手近战）
  专属职业能力（4.9 节，需 JP 学习 + 职业 × 武器匹配）：
    - 穿心刺（匕首 + 斥候/不良人...）
    - 横扫千军（陌刀 + 陌刀手）
    - 剑舞（剑 + 诗剑仙）
    - 抵近射击（擘张弩 + 弩手）
    - 碎甲、击晕、重击、裂盾、放血……（详见 4.9.3 节 / class-system.md）

Step 2：命中检定（减法制）
  Hit% = attacker_hit
       - eff_def
       + 围攻奖励（每多一友军 +5%，最多 +20%）
       + 高地修正（高 +10% / 低 -10%）
       + 距离修正
       + 职业能力命中修正（精准攻击 +15 / 全力一击 -10% / 穿心刺 -15% / ...）
       + 攻击类型修正（Pierce -5% / 匕首额外 -5%）
       + 士气 / 气力等其他修正
  
  if rand(0,100) >= Hit% → MISS

Step 3：格挡检定（独立概率）
  block_chance = weapon.block_value + shield_bonus
  if rand(0,100) < block_chance → BLOCK（伤害归零）

Step 4：部位检定
  普攻：单手武器 rand(0,100) < 25 → 头部；双手武器 rand(0,100) < 15 → 头部；否则身体
  精准攻击（近战）：rand(0,100) < 55 → 头部，否则身体
  精准攻击（远程）：rand(0,100) < 50 → 头部，否则身体
  全力一击：与普攻相同（双手 15% 头部，不提升）
  穿心刺 Puncture：强制身体（不掷头部检定）
  其他专属职业能力：默认按武器普攻部位概率，特例由能力定义说明

Step 5：暴击检定（独立于部位）
  crit_chance = 5（基础）+ 武器暴击 + 词条 + 天赋
  if rand(0,100) < crit_chance → 暴击

Step 6：基础伤害计算
  base = damage_base
       × 气力档位系数（>50%:1.0 / 20-50%:0.85-0.95 / 1-20%:0.65-0.85 / 力竭:0.5）
       × 微小波动 random(0.95, 1.05)
       × 职业专精系数（不擅长 0.9 / 合格 1.0 / 擅长 1.10 / 精通 1.20）
       × (1 + 词条加成总和)（≤ +30%）
       × 职业能力伤害修正（普攻 ×1.0 / 瞄头 ×1.10 / 全力一击 ×1.15 / 精准射击 ×1.0 / 各专属能力按其定义）
       × 双手握持修正（单手武器空副手时 ×1.25 / 否则 ×1.0）
  
  // 关键约束：4.8 节"一次攻击只能选 1 个职业能力" → 职业能力修正只取 1 个来源
  
Step 7：暴击 + 头部加法叠加（核心）
  multiplier = 1.0
  if 头部命中：multiplier += 0.5
  if 暴击：multiplier += 0.5
  
  最终伤害 = base × multiplier
  // 不再有 hard cap 截断；约束由 4.8 节互斥规则 + 触发概率 + 词条上限自然提供

Step 8：伤害分配
  // === 8.0 特殊分支：穿心刺（专属职业能力，匕首专精）===
  if 职业能力 == 穿心刺 Puncture：
    护甲扣 = 0  // 完全无视护甲
    HP 扣 = 最终伤害（100% 转 HP）
    跳过 Step 8 后续步骤
  
  // === 8.1 普通分配（基于攻击类型 Slash / Pierce / Crush）===
  目标护甲 = 头部命中时用 Head Armor，否则 Body Armor
  
  if 护甲 > 0：
    应扣护甲 = 最终伤害 × armor_mult  (Slash 1.0 / Pierce 0.7 / Crush 1.5)
    
    // 渗透 HP 公式：基础渗透率 × weight_modifier
    基础渗透率 = (Slash 0.10 / Pierce 0.15 / Crush 0.08)  ← 攻击类型决定
    weight_modifier = 1.0 + max(0, (weapon.weight - 4) / 6)  ← 武器重量决定
    渗透 HP = 最终伤害 × 基础渗透率 × weight_modifier
    
    // 暴击的攻击类型差异化（暴击基础率翻倍）
    if 暴击：基础渗透率 ×= 2.0
      → Slash 暴击 0.20 / Pierce 暴击 0.30 / Crush 暴击 0.16
    if Crush 且 暴击：应扣护甲 += 30  // "震碎甲叶" 额外效果保留
    
    // 重新计算暴击渗透
    渗透 HP = 最终伤害 × 基础渗透率（含暴击翻倍）× weight_modifier
    
    // 专属职业能力对护甲处理的修正（举例，详见 class-system.md）
    if 职业能力 == 碎甲：armor_mult ×= 1.7  // Crush 1.5 → 2.55
    if 职业能力 == 锐不可当：在 Step 3 阶段 block_contribution ×= 0.5
  
  // 8.2 处理护甲伤害溢出（破甲爆发）
  if 应扣护甲 > 当前护甲：
    实际护甲扣 = 当前护甲（清零）
    溢出量 = 应扣护甲 - 当前护甲
    溢出 HP = 溢出量 × hp_mult  (Slash 1.0 / Pierce 1.0 / Crush 0.7)
  else：
    实际护甲扣 = 应扣护甲
    溢出 HP = 0
  
  // 8.3 总 HP 损失 = 渗透 HP + 溢出 HP
  总 HP 扣 = 渗透 HP + 溢出 HP
  
  // 8.4 如果攻击命中时护甲已经为 0
  if 护甲 = 0（攻击前已破）：
    HP 扣 = 最终伤害 × hp_mult  (Slash 1.0 / Pierce 1.0 / Crush 0.7)
    护甲扣 = 0
    
  // 关键设计：破甲爆发瞬间
  // 当一击刚好破完护甲时，剩余的破甲伤害"溢出"为 HP 伤害（× hp_mult）
  // 这让"破甲一击"成为战术爆点，而不是浪费的破甲值

Step 9：气力消耗（基础 + 职业能力附加，不显式给玩家）
  本次攻击气力消耗 = base_stamina × weight_mult
                + 职业能力隐藏气力消耗（普攻 0 / 瞄头 +10 / 精准 +10 / 全力一击 +12 / 穿心刺 +12 /
                                  碎甲 +12 / 击晕 +15 / 重击 +20 / 裂盾 +15 / 放血 +12 / ...）

Step 10：全力一击防御代价（双向特技代价）
  if 职业能力 == 全力一击：
    本回合自身近战防御 -10（直到本回合结束）
    // 物理意义：挥砍轨迹大、露出破绽
```

### 8.2 关键设计原则

1. **传说不一定比优秀绝对强**——"上限更高、特化更极端"
   - 优秀武器 = 稳定可靠的好搭档（中等加成 + 轻度副作用）
   - 传说武器 = 极端特化的怪物（高加成 + 重度副作用，仅在适配 build 下超越优秀）
2. **主力是优秀级**——玩家整局 80% 时间用优秀武器
3. **传说获取多元**：
   - **锻造**（高成本）：消耗稀有材料 + 钱 + 时间
   - **故事事件**：主线/支线触发的剧情奖励（带 origin_story 字段）
   - **野外随机 Boss**：大地图随机刷新的精英怪掉落
   - **战场宝箱**：特定战斗中的隐藏宝箱（与侦察系统联动？）
4. **不抢人物培养风头**：词条加成总和 ≤ +30%，专精系数（×1.2）才是更大的 bonus
5. **每件武器词条不重复**

| **持重** | 站定不动 1 回合后 +10 防 | 移动后下次格挡 -10 | 守阵兵种 |

#### 操作系（节奏风格）

| 词条 | 正向效果 | 负向代价 | 适配 build |
|---|---|---|---|
| **流畅** | 本武器 AP -1 | 本武器伤害 -10% | 高频攻击 build |
| **凝神** | 瞄头惩罚 -10 | 普攻命中 -5 | 瞄头流刺客/弓手 |
| **疾风** | Init +10 | 最大 HP -10 | 速度流游侠 |
| **沉重** | 命中击退 1 格 | AP +1 | 控制流战锤 |
| **延迟** | 蓄力 1 回合后下击伤害 +50% | 立即攻击伤害 -10% | 一回合爆发流 |

#### 特色系（极稀有，传说级专属）

| 词条 | 正向效果 | 负向代价 | 备注 |
|---|---|---|---|
| **千钧一发** | HP < 30% 时伤害 +30% | HP > 70% 时无加成 | 残血激发 |
| **神来之笔** | 每场战斗第 1 击必命中 + 暴击 | 第 1 击之外暴击率 -5 | 单次爆发 |
| **精通** | 持有者视为本武器精通 | 仅本武器有效，换武器失效 | **build 改变神技** ⭐ |
| **附魂** | 杀敌后获得 +1 melee 永久（仅本场战斗）| 角色阵亡时立即清空 | 战斗内成长流 |

### 8.4 词条强度约束（红线落地）

- 单件武器词条加成总和 ≤ +30% 武器伤害（含正向词条）
- 双向词条的"负向代价"不计入 +30% 上限（鼓励双向设计）
- 特色系词条每场战斗仅触发 1 次
- 同一武器内词条不重复

### 8.5 词条获取与重铸（Phase 3+ 详定）

- **锻造**：传说武器锻造时词条随机生成（XCOM 派）
- **重铸**（待定）：消耗稀有材料 + 钱，重新随机词条
- **拒绝词条交易**：不允许玩家"刻字定制"特定词条，保持随机性的乐趣
- **传说武器叙事**：每件传说武器有 `origin_story` 字段，记录获取剧情

## 九、武器与气力消耗的关系

复用 design.md "气力系统"小节的统一规则：

```
任何动作消耗 = ceil(base_cost × weight_mult)
weight_mult = 1 + (角色装备总 weight) × 0.02
```

武器的 `weight` 字段直接叠加到角色装备总 weight。例：
- 跳荡装备横刀(4) + 团牌(2) + 中甲(18) → 总 weight 24 → weight_mult 1.48
- 陌刀手装备陌刀(9) + 重甲(30) → 总 weight 39 → weight_mult 1.78（陌刀手气力消耗最高）
- 刺客装备双匕首(2) + 轻甲(8) → 总 weight 10 → weight_mult 1.20

→ 重武器 + 重甲 = 气力消耗成倍上升，自然限制"重甲陌刀手暴打全场"

## 十一、命名规范与 ID 约定

- 普通武器 `id` = 武器英文名（如 `saber`、`spear`、`crossbow`）
- 优秀/传说武器 `id` = `{base_id}_{descriptor}_{number}`，并保留 `base_id` 字段指向基础武器
- `name` 用唐代名，UI 显示用
- `affixes[].id` 用 snake_case 英文（便于代码引用）
- `affixes[].name` 用中文（玩家展示）

design.md 的"战斗模型 v3 第五节：武器分类型"将简化为：

```
### 五、武器系统（详见 design/weapon-system.md）

核心要点摘要：
- 精简武器属性，取消 damage min/max，改用 damage_base + 气力档位

- 双手握持（Double Grip）：单手武器空副手时 +25% 伤害（BB 锚点），物理逻辑是辅助手提供力矩加成；防御代价是失去盾牌/副武器格挡

- 暴击 + 头部加法叠加（×0.5 + ×0.5 = ×2.0），无 cap 截断；约束由互斥规则 + 触发概率 + 词条上限（≤ +30%）自然提供
- Pierce 武器统一 -5% 命中；瞄头惩罚与武器重量挂钩

- 三档稀有度：普通 / 优秀 / 传说，双向词条（火纹派）确保武器特化但不超模
- **互斥规则统一**：一次攻击仅 1 个职业能力（通用 OR 专属）
```
