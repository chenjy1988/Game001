# Battle for Wesnoth Terrain Tiles

## Source
本目录下所有 PNG 资源来自开源游戏 **Battle for Wesnoth**，
仓库：https://github.com/wesnoth/wesnoth

具体路径来自：`data/core/images/terrain/`，包括：
- `grass/green*.png` `grass/semi-dry*.png` `grass/leaf-litter*.png`
- `flat/dirt*.png` `flat/dirt-dark*.png`
- `sand/beach*.png`

## Tile 规格
- 尺寸：72×72 px
- 格式：RGBA PNG（透明背景，hex 形状已裁好）
- 朝向：pointy-top（与本项目 hex grid 一致）
